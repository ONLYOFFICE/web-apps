Visualizador de apresentações

Quando você abre o Editor de apresentações do ONLYOFFICE com direitos de acesso de visualizador, você pode navegar livremente pela apresentação usando as ferramentas de visualização disponíveis.

  1. A barra de ferramentas superior tem as seguintes guias e botões de acesso rápido:
    • Guias disponíveis: guia Arquivo, guia Exibir, guia Plugins, guia IA.

      A guia Exibir permite alterar o estilo do ponteiro. Use a ferramenta Mão para navegar no slide, e a ferramenta Selecionar para selecionar objetos do slide.

    • Botões de acesso rápido:
      • Editar o arquivo atual - feche o modo de visualização da apresentação e mude para a edição.
      • Imprimir - imprimir a apresentação atual.
      • Abrir local do arquivo - na versão para desktop, permite abrir a pasta onde o arquivo está armazenado, na janela do Explorador de arquivos. Na versão online, permite abrir a pasta do arquivo do módulo Documentos, onde o arquivo está armazenado, em uma nova guia do navegador.
      • Baixar arquivo - faça o download da apresentação atual em sua unidade de disco rígido.
      • Pesquisa - permite pesquisar na apresentação uma palavra ou um símbolo específico, etc.
  2. O painel esquerdo tem os seguintes botões:
    • - permite usar a ferramenta Pesquisar e substituir,
    • - permite visualizar slides e navegar por eles,
    • - permite abrir o painel Comentários,
    • - (disponível apenas na versão online only) permite abrir o painel Bate-papo,
    • - (disponível apenas na versão online) permite entrar em contato com nossa equipe de suporte,
    • - (disponível apenas na versão online) permite visualizar as informações sobre o programa.
  3. A Lista de slides tem todos os slides da apresentação disponíveis para acesso rápido.
  4. A Barra de status na parte inferior da janela do editor contém o ícone Iniciar apresentação de slides, algumas ferramentas de navegação: indicador do número do slide e botões de zoom. A Barra de status também exibe algumas notificações (tais como "Todas as alterações salvas", ‘Perda de conexão’ quando não há conexão e o editor está tentando se reconectar etc).
  5. A Barra de rolagem à direita permite rolar a apresentação para cima e para baixo.
