Pixabay

O plug-in Pixabay permite que você adicione imagens à sua apresentação a partir de uma coleção aberta do serviço Pixabay, que oferece imagens livres de direitos autorais.

O plug-in é compatível com as versões auto-hospedada e a para desktop dos editores do ONLYOFFICE e podem ser adicionados às instâncias do ONLYOFFICE manualmente usando o Gerenciador de plug-ins.

Instalação

Para instalar o plug-in Pixabay,

  1. Vá para a guia Plug-ins.
  2. Abra o Gerenciador de plug-ins.
  3. Ache
    no mercado e clique no botão Instalar abaixo.
  4. Clique no ícone
    Pixabay na guia Plug-ins.
  5. Prossiga com a configuração do plug-in.

Para obter mais detalhes, consulte a documentação do ONLYOFFICE API.

Configuração

  1. Faça login na sua conta do Pixabay ou registre uma nova conta.
  2. Vá para a seção Pesquisar Imagens da página do Pixabay API.
  3. Role para baixo até a lista de Parâmetros e copie o link no parâmetro Chave. Se você não estiver conectado, pressione Login ao lado do parâmetro Chave.

  4. Cole a chave no campo Chave da API no painel esquerdo da guia Plug-ins no Editor de apresentações.
  5. Clique em Salvar.

Como usar

  1. Vá para a guia Plug-ins.
  2. Clique no ícone
    Pixabay.
  3. No painel esquerdo que se abre, digite a palavra-chave que você associa à imagem que está procurando.
  4. Use o botão Exibir configurações para refinar sua pesquisa por Idioma, Tipo de imagem, Orientação e Categoria. Clique em Ocultar configurações para reduzir a seção de pesquisa.

  5. Clique em Pesquisa.
  6. Percorra os resultados da pesquisa e clique na imagem para adicioná-la à sua apresentação.
