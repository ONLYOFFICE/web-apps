Zoom

Mit dem Zoom-Plugin können Sie Zoom-Meetings mit nur wenigen Klicks direkt im Editor veranstalten oder planen.

Das Plugin ist mit der selbst gehosteten Version der ONLYOFFICE-Editoren kompatibel und kann manuell zu ONLYOFFICE-Instanzen hinzugefügt werden. Sie können auch das Plugin mit der Plugin-Verwaltung installieren.

Installation

Um das Zoom-Plugin zu installieren,

  1. Gehen Sie zur Registerkarte Plugins.
  2. Öffnen Sie die Plugin-Verwaltung.
  3. Suchen Sie
    Zoom auf dem Marktplatz und klicken Sie unten auf die Schaltfläche Installieren.
  4. Klicken Sie auf der Registerkarte Plugins auf das Symbol
    Zoom.
  5. Fahren Sie mit der Plugin-Konfiguration fort.

Weitere Informationen finden Sie in der ONLYOFFICE-API-Dokumentation.

Konfiguration

  1. Registrieren Sie den Editor als JWT-App auf der Seite Zoom Develop, um den Meeting-Plan zu verwenden. Füllen Sie alle erforderlichen Felder aus und aktivieren Sie die App. Der App wird ein JWT-Token zugewiesen.

    Wenn Sie Hilfe beim Ausfüllen der Felder benötigen, lesen Sie bitte die offizielle Zoom-Dokumentation.

  2. Erstellen Sie eine Meeting SDK-App auf der Seite Zoom Develop , um mit der Teilnahme an Meetings zu beginnen. Füllen Sie alle erforderlichen Felder aus und aktivieren Sie die App. SDK-Anmeldedaten werden der App zugewiesen.

    Wenn Sie Hilfe beim Ausfüllen der Felder benötigen, lesen Sie bitte die offizielle Zoom-Dokumentation.

  3. Geben Sie den generierten SDK-Schlüssel, SDK-Secret und JWT-Token in die entsprechenden Felder im linken Bereich des ONLYOFFICE-Editors ein und klicken Sie auf Speichern.

Verwendung

  1. Gehen Sie zur Registerkarte Plugins.
  2. Klicken Sie auf das Symbol
    Zoom.
  3. Fügen Sie Ihr Besprechungsthema hinzu und wählen Sie aus, ob Sie Start a meeting oder Schedule a meeting möchten.
  4. Meeting-Benachrichtigungen werden für Online-Editoren an den Chat und für Desktop-Editoren an die Kommentare gesendet.

Beim Beitritt zu einem Meeting kann im Safari-Browser ein schwarzes Fenster erscheinen. Damit das Problem verschwindet, müssen Sie die Größe des Plugin-Fensters ändern oder die Browserseite vergrößern.
