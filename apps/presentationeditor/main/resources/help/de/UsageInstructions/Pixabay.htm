Pixabay

Mit dem Plugin Pixabay können Sie Ihrer Präsentation Bilder aus einer offenen Sammlung des Pixabay-Dienstes hinzufügen, der urheberrechtsfreie Bilder anbietet.

Das Plugin ist mit den selbstgehosteten und den Desktop-Versionen der ONLYOFFICE-Editoren kompatibel und kann mithilfe des Plugin-Managers manuell zu ONLYOFFICE-Instanzen hinzugefügt werden.

Installation

Um das Pixabay-Plugin zu installieren,

  1. Gehen Sie zur Registerkarte Plugins.
  2. Öffnen Sie den Plugin-Manager.
  3. Suchen Sie im Marktplatz nach
    und klicken Sie unten auf die Schaltfläche Installieren.
  4. Klicken Sie auf der Registerkarte Plugins auf das
    -Symbol Pixabay.
  5. Fahren Sie mit der Plugin-Konfiguration fort.

Lesen Sie bitte für weitere Details die ONLYOFFICE API-Dokumentation.

Konfiguration

  1. Melden Sie sich bei Ihrem Pixabay-Konto an oder registrieren Sie ein neues.
  2. Gehen Sie zum Abschnitt Bilder suchen der Pixabay API-Seite.
  3. Scrollen Sie nach unten zur Liste Parameter und kopieren Sie den Link unter dem Parameter Schlüssel. Wenn Sie nicht angemeldet sind, drücken Sie neben dem Parameter Schlüssel auf Anmelden.

  4. Fügen Sie den Schlüssel in das Feld API-Schlüssel im linken Bereich der Registerkarte Plugins in der Präsentationseditor ein.
  5. Klicken Sie auf Speichern.

Anwendung

  1. Gehen Sie zur Registerkarte Plugins.
  2. Klicken Sie auf das
    Pixabay-Symbol.
  3. Geben Sie im sich öffnenden linken Bereich das Schlüsselwort ein, das Sie mit dem gesuchten Bild verknüpfen.
  4. Verwenden Sie die Schaltfläche Einstellungen anzeigen, um Ihre Suche nach Sprache, Bildtyp, Ausrichtung und Kategorie zu verfeinern. Klicken Sie auf Einstellungen ausblenden, um den Suchbereich einzuschränken.

  5. Klicken Sie auf Suchen.
  6. Scrollen Sie durch die Suchergebnisse und klicken Sie auf das Bild, um es Ihrer Präsentation hinzuzufügen.
