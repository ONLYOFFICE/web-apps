Unterstützte Formate elektronischer Präsentationen

Eine Präsentation besteht aus einer Reihe von Folien, die verschiedene Arten von Inhalten enthalten können, z. B. Bilder, Mediendateien, Text, Effekte usw. Der Präsentationseditor unterstützt die folgenden Formate:

Beim Hochladen oder Öffnen der Datei für die Bearbeitung wird sie ins Office-Open-XML-Format (PPTX) konvertiert. Dies wird gemacht, um die Dateibearbeitung zu beschleunigen und die Interfunktionsfähigkeit zu erhöhen.

Die folgende Tabelle enthält die Formate, die zum Anzeigen und/oder zur Bearbeitung geöffnet werden können.

Formate Beschreibung Nativ anzeigen Anzeigen nach Konvertierung in OOXML Nativ bearbeiten Bearbeitung ach Konvertierung in OOXML
DPS WPS Office-Präsentationsdokument von Kingsoft
Ein im WPS Office-Paket enthaltenes Diashow-Präsentationsformat, das eine Sammlung von Folien enthält, die aus einer Titelfolie gefolgt von Inhaltsfolien bestehen kann, die Text, Bilder, Videos, Audio und Formen enthalten können.		 + +
DPT Kingsoft-Präsentationsvorlage
Ein Dateiformat zum Speichern des Standardlayouts und der Standardstile einer Präsentation und zum Erstellen von Präsentationen (.DPS-Dateien) mit derselben Formatierung.		 + +
FODP OpenDocument Flat XML-Präsentationsdateiformat
Ein Dateiformat zum Speichern von Präsentationen im OpenDocument-XML-Dateiformat ohne Komprimierung.		 + +
ODP OpenDocument Präsentation
Dateiformat, das mit der Impress-Anwendung erstellte Präsentationen darstellt; diese Anwendung ist ein Bestandteil des OpenOffice-Pakets.		 + +
OTP OpenDocument-Präsentationsvorlage
OpenDocument-Dateiformat für Präsentationsvorlagen. Eine OTP-Vorlage enthält Formatierungseinstellungen, Stile usw. und kann zum Erstellen mehrerer Präsentationen mit derselben Formatierung verwendet werden.		 + +
POT Portable Objektvorlage
Ein Dateiformat zum Erstellen von Diashow-Präsentationen, das das Standardlayout, die Formatierung und die Stile für eine Diashow enthält. POT-Dateien werden verwendet, um mehrere PPT-Dateien mit derselben Formatierung zu erstellen.		 + +
POTM Eine makrofähige Präsentationsvorlage von Microsoft PowerPoint
Ein Dateiformat zum Erstellen von Diashow-Präsentationen, das Standardbilder, Folienvorlagen, Makros und Formatierungen enthält.		 + +
POTX PowerPoint Office Open XML Dokumenten-Vorlage
Gezipptes, XML-basiertes, von Microsoft für Präsentationsvorlagen entwickeltes Dateiformat. Eine POTX-Vorlage enthält Formatierungseinstellungen, Stile usw. und kann zum Erstellen mehrerer Präsentationen mit derselben Formatierung verwendet werden.		 + +
PPS PowerPoint-Diashow-Datei
Ein Dateiformat, das bei der Vorbereitung von Diashows und Präsentationen verwendet wird, um verschiedene Daten wie Video, Audio, Text, Animationen und Bilder zu speichern.		 + +
PPSM Makrofähige Diashow von Microsoft PowerPoint
Ein Dateiformat, das zum Erstellen von Diashows und Präsentationen verwendet wird und eine oder mehrere Folien mit Makros enthält, die Text, Bilder und Übergänge enthalten können.		 + +
PPSX Microsoft PowerPoint-Diashow
Präsentationsdateiformat, das für die Wiedergabe von Slideshows verwendet wird.		 + +
PPT Dateiformat, das in Microsoft PowerPoint verwendet wird. + +
PPTX Office Open XML Präsentation
Gezipptes, XML-basiertes, von Microsoft entwickeltes Dateiformat zur Präsentation von Kalkulationstabellen, Diagrammen, Präsentationen und Textverarbeitungsdokumenten		 + +
SXI StarOffice Impress Präsentation
Ein Präsentationsformat zum Erstellen von Diashow-Präsentationen, die aus Titel- und Inhaltsfolien mit Text und digitalen Objekten bestehen können; unterstützt Audio-, Video- und Übergangseffekte.		 +
