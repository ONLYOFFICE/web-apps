Tastenkombinationen für Key-Tipps

Verwenden Sie Tastenkombinationen für einen schnelleren und einfacheren Zugriff auf die Funktionen des Präsentationseditors ohne eine Maus zu verwenden.

  1. Drücken Sie die Alt-Taste, um alle wichtigen Tipps für die Kopfzeile des Editors, die obere Symbolleiste, die rechte und linke Seitenleiste und die Statusleiste einzuschalten.
  2. Drücken Sie den Buchstaben, der dem Element entspricht, das Sie verwenden möchten. Die zusätzlichen Tastentipps können je nach gedrückter Taste angezeigt werden. Die ersten Tastentipps werden ausgeblendet, wenn zusätzliche Tastentipps angezeigt werden.

    Um beispielsweise auf die Registerkarte Einfügen zuzugreifen, drücken Sie Alt, um alle Tipps zu den Primärtasten anzuzeigen.

    Drücken Sie den Buchstaben I, um auf die Registerkarte Einfügen zuzugreifen, und Sie sehen alle verfügbaren Verknüpfungen für diese Registerkarte.

    Drücken Sie dann den Buchstaben, der dem zu konfigurierenden Element entspricht.

  3. Drücken Sie Alt, um alle Tastentipps auszublenden, oder drücken Sie Escape, um zur vorherigen Gruppe von Tastentipps zurückzukehren.

Bitte beachten: Unter macOS enthalten einige Tastenkombinationen die Tasten POS1, ENDE, Bild auf und Bild ab, die nur auf der erweiterten Tastatur verfügbar sind. and others		 Bei Verwendung der Tastenkombinationen ⌥ Option+Tastatursymbol werden für Tastaturen, die das Einfügen von Unicode-Symbolen unterstützen, die Tastatursymbole hinzugefügt. Die Beispiele sind unten aufgeführt.
Beim englischen ABC-Layout fügt die Tastenkombination ⌥ Option+Q das Symbol „œ“ ein, die Tastenkombination ⌥ Option+F fügt das Funktionssymbol „ƒ“ ein.
Beim US-International-Layout ohne Deadkeys fügt die Tastenkombination ⌥ Option+Q das Symbol „ä“ ein.
Beim schweizerisch-deutschen Layout fügt die Tastenkombination ⇧ UMSCHALT+⌥ Option+7 das Symbol „\“ ein.
Im Text navigieren
Ein Zeichen nach links/rechts oder eine Zeile nach oben/unten bewegen ← → ↑ ↓ ← → ↑ ↓ Bewegen Sie den Cursor ein Zeichen nach links/rechts oder eine Zeile nach oben/unten.
Zum Anfang eines Wortes oder ein Wort nach links gehen STRG+← ⌥ Option+← Bewegen Sie den Cursor an den Anfang eines Wortes oder ein Wort nach links.
Ein Wort nach rechts verschieben STRG+→ ⌥ Option+→ Bewegen Sie den Cursor ein Wort nach rechts.
Zum nächsten Platzhalter wechseln oder neue Folie erstellen STRG+↵ Eingabetaste ^ STRG+↵ Eingabetaste,
⌘ Cmd+↵ Eingabetaste		 Zum nächsten Titel- oder Textkörperplatzhalter wechseln. Wenn es sich um den letzten Platzhalter auf einer Folie handelt, wird eine neue Folie mit demselben Folienlayout wie die ursprüngliche Folie eingefügt.
Zum Zeilenanfang springen POS1 ⌘ Cmd+←

POS1		 Setzen Sie den Cursor an den Anfang der aktuell bearbeiteten Zeile.
Zum Ende der Zeile springen ENDE ⌘ Cmd+→

ENDE		 Setzen Sie den Cursor an das Ende der aktuell bearbeiteten Zeile.
Zum Anfang des Inhalts springen STRG+POS1 ^ STRG+Fn+←

^ STRG+POS1		 Setzen Sie den Cursor an den Anfang des aktuell bearbeiteten Textfelds oder in die obere linke Zelle einer Tabelle.
Zum Ende des Inhalts springen STRG+ENDE ^ STRG+Fn+→

^ STRG+ENDE		 Setzen Sie den Cursor an das Ende des aktuell bearbeiteten Textfelds oder in die untere rechte Zelle einer Tabelle.

Fügen Sie die kopierten Daten mit Strg+V unter Windows bzw. Cmd+V unter macOS ein. Öffnen Sie nach dem Einfügen der kopierten Daten mit der Taste Strg das Menü Spezielles einfügen und drücken Sie dann die Buchstabentaste, die der gewünschten Option entspricht.
