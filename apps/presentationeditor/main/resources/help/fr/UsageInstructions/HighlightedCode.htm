Insérer le code en surbrillance

Dans l'éditeur de présentations, vous pouvez intégrer du code mis en surbrillance avec un style déjà appliqué selon le langage de programmation et le style de couleur dans l'application choisie.

À partir de la version 8.2 ONLYOFFICE Docs, aucun module complémentaire n'est inclus dans l'éditeur par défaut. Il faut installer les modules complémentaires à l'aide du Gestionnaire de plugins.

  1. Accédez à votre présentation et placez le curseur à l'endroit où le code doit être inséré.
  2. Passez à l'onglet Modules complémentaires et choisissez
    Code en surbrillance.
  3. Spécifiez la Langue de programmation.
  4. Choisissez le Style du code pour qu'il apparaisse de manière à sembler celui dans le programme.
  5. Spécifiez si on va remplacer les tabulations par des espaces.
  6. Choisissez la Couleur de fond. Pour ce faire, déplacez manuellement le curseur sur la palette de couleurs ou ajoutez la valeur de type RBG/HSL/HEX.
  7. Cliquez sur OK pour insérer le code.
