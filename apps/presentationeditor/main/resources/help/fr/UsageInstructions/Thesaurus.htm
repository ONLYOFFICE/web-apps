Remplacer un mot par synonyme

Si on répète plusieurs fois le même mot ou si un mot ne vous convient pas, l'éditeur de présentations ONLYOFFICE vous permet de rechercher des synonymes. Retrouvez les antonymes du mot affiché aussi.

À partir de la version 8.2 ONLYOFFICE Docs, aucun module complémentaire n'est inclus dans l'éditeur par défaut. Il faut installer des modules complémentaires à l'aide du Gestionnaire de plugins.

  1. Sélectionnez le mot dans votre présentation.
  2. Passez à l'onglet Modules complémentaires et sélectionnez
    Thésaurus.
  3. Le panneau gauche liste les synonymes et les antonymes.
  4. Cliquez sur le mot remplaçant à insérer dans votre présentation.
