Remplacer un mot par synonyme

Si on répète plusieurs fois le même mot ou il ne semble pas que le mot est juste, ONLYOFFICE Presentation Editor vous permet de trouver les synonymes. Retrouvez les antonymes du mot affiché aussi.

  1. Sélectionnez le mot dans votre présentation.
  2. Passez à l'onglet Modules complémentaires et choisissez Thésaurus.
  3. Le panneau gauche liste les synonymes et les antonymes.
  4. Cliquez sur le mot à remplacer dans votre document.
