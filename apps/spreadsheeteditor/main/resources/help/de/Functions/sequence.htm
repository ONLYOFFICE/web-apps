SEQUENZ-Funktion

Die SEQUENZ-Funktion ist eine der Mathematik und Trigonometrie-Funktionen. Sie wird verwendet, um eine Abfolge von Zahlen zurückzugeben.

Bitte beachten Sie, dass dies eine Matrixformel ist. Um mehr zu erfahren, lesen Sie bitte den Artikel Matrixformeln einfügen.

Die Formelsyntax der Funktion SEQUENZ ist:

SEQUENZ(Zeilen,[Spalten],[Anfang],[Schritt])

dabei gilt

Zeilen wird verwendet, um die Anzahl der zurückzugebenden Zeilen festzulegen.

Spalten wird verwendet, um die Anzahl der zurückzugebenden Spalten festzulegen.

Anfang wird verwendet, um die erste Zahl in der Sequenz festzulegen.

Schritt wird verwendet, um das Inkrement für jeden nachfolgenden Wert im Array zu steuern.

Anwendung der Funktion SEQUENZ:

  1. Wählen Sie die Zelle aus, in der Sie das Ergebnis anzeigen möchten.
  2. Klicken Sie auf das Symbol Funktion
    in der oberen Symbolleiste
    oder klicken Sie mit der rechten Maustaste in eine ausgewählte Zelle und wählen Sie die Option Funktion einfügen aus dem Menü
    oder klicken Sie auf das Symbol
    in der Bearbeitungsleiste.
  3. Wählen Sie aus der Liste die Funktionsgruppe Mathematik und Trigonometrie.
  4. Klicken Sie auf die Funktion SEQUENZ.
  5. Geben Sie die erforderlichen Argumente ein, indem Sie sie durch Kommas trennen.
  6. Drücken Sie die Eingabe-Taste.

Das Ergebnis wird in der ausgewählten Zelle angezeigt.
