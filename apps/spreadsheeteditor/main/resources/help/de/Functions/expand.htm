ERWEITERN-Funktion

Die ERWEITERN-Funktion ist eine der Suchen- und Bezügefunktionen. Sie wird verwendet, um eine Matrix auf die angegebenen Dimensionen zu erweitern.

Bitte beachten Sie, dass dies eine Matrixformel ist. Um mehr zu erfahren, lesen Sie bitte den Artikel Matrixformeln einfügen.

Die Formelsyntax der Funktion ERWEITERN ist:

=ERWEITERN(array, rows, [columns], [pad_with])

dabei gilt

array ist der Zellbereich, der erweitert werden soll.

rows ist die Anzahl der Zeilen im zurückgegebenen Array. Obwohl dies ein erforderliches Argument ist, werden keine Zeilen hinzugefügt, wenn es weggelassen wird, und die Anzahl der Spalten muss angegeben werden.

[columns] ist ein optionales Argument, das die Anzahl der Spalten im zurückgegebenen Array angibt. Wenn es weggelassen wird, werden keine Spalten hinzugefügt und die Anzahl der Zeilen muss angegeben werden.

[pad_with] ist der Wert, der in die hinzugefügten Zellen eingefügt werden soll. #N/A ist der Standardwert.

Anwendung der Funktion ERWEITERN:

  1. Wählen Sie die Zelle aus, in der Sie das Ergebnis anzeigen möchten.
  2. Klicken Sie auf das Symbol Funktion
    in der oberen Symbolleiste
    oder klicken Sie mit der rechten Maustaste in eine ausgewählte Zelle und wählen Sie die Option Funktion einfügen aus dem Menü
    oder klicken Sie auf das Symbol
    in der Bearbeitungsleiste.
  3. Wählen Sie aus der Liste die Funktionsgruppe Suchen und Bezüge.
  4. Klicken Sie auf die Funktion ERWEITERN.
  5. Geben Sie die erforderlichen Argumente ein, indem Sie sie durch Kommas trennen.
  6. Drücken Sie die Eingabe-Taste.

Das Ergebnis wird in der ausgewählten Zelle angezeigt.
