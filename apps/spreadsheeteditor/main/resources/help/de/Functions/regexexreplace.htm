Funktion REGEXERSETZEN

Die Funktion REGEXERSETZEN ist eine der Text- und Datenfunktionen. Sie wird verwendet, um Text innerhalb einer Zeichenfolge, der einem Muster eines regulären Ausdrucks entspricht, durch einen bestimmten Ersetzungstext zu ersetzen.

Syntax

REGEXERSETZEN(text; muster; ersatz_text; [vorkommen]; [beachten_groß_klein])

Die Funktion REGEXERSETZEN hat folgende Argumente:

Argument Beschreibung
text Die Textzeichenfolge oder der Verweis auf eine Zelle, die den Text enthält, in dem Sie Ersetzungen vornehmen möchten.
muster Der reguläre Ausdruck („Regex"), der das Textmuster beschreibt, das Sie ersetzen möchten.
ersatz_text Der Text, den Sie für das übereinstimmende Muster einsetzen möchten. Sie können $1, $2 usw. verwenden, um auf Erfassungsgruppen aus dem Muster zu verweisen.
vorkommen Gibt an, welche Instanz des Musters ersetzt werden soll. Dies ist ein optionales Argument. Wenn es weggelassen wird, ist der Standardwert 0, wodurch alle Instanzen ersetzt werden. Eine positive Zahl ersetzt das jeweilige Vorkommen. Eine negative Zahl ersetzt Vorkommen vom Ende her gezählt.
beachten_groß_klein Legt fest, ob beim Abgleich die Groß-/Kleinschreibung beachtet wird. Dies ist ein optionales Argument. Wenn es weggelassen wird, wird die Groß-/Kleinschreibung beachtet (0). Geben Sie 0 für einen Abgleich mit Berücksichtigung der Groß-/Kleinschreibung oder 1 für einen Abgleich ohne Berücksichtigung der Groß-/Kleinschreibung ein.

Hinweise

Beim Schreiben von Regex-Mustern können Symbole, sogenannte „Tokens", verwendet werden, die mit einer Vielzahl von Zeichen übereinstimmen. Im Folgenden finden Sie einige einfache Tokens zur Referenz:

  • [0-9]: eine beliebige Ziffer
  • [a-z]: ein Zeichen im Bereich von a bis z
  • [A-Z]: ein Zeichen im Bereich von A bis Z
  • .: ein beliebiges einzelnes Zeichen
  • *: null oder mehr des vorangehenden Zeichens
  • +: ein oder mehr des vorangehenden Zeichens
  • (): Erfassungsgruppe – erfasster Text kann in der Ersetzung als $1, $2 usw. referenziert werden
  • {n}: genau n Vorkommen des vorangehenden Zeichens
  • \: Escapezeichen für Sonderzeichen

Erfassungsgruppen sind Teile eines Regex-Musters, die von Klammern „(...)" umgeben sind. Im Argument Ersetzung können Sie auf erfasste Gruppen verweisen, indem Sie Folgendes verwenden:

  • $1 – erste Erfassungsgruppe
  • $2 – zweite Erfassungsgruppe
  • $n – die n-te Erfassungsgruppe

Die Funktion gibt die geänderte Textzeichenfolge zurück. Der ursprüngliche Text in der Quellzelle wird nicht geändert.
