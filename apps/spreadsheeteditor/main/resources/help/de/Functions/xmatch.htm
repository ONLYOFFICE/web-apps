XVERGLEICH-Funktion

Die XVERGLEICH-Funktion ist eine der Suchen- und Bezügefunktionen. Sie wird verwendet, um die relative Position eines Elements in einem Array zurückzugeben. Standardmäßig ist eine genaue Übereinstimmung erforderlich.

Bitte beachten Sie, dass dies eine Matrixformel ist. Um mehr zu erfahren, lesen Sie bitte den Artikel Matrixformeln einfügen.

Die Formelsyntax der Funktion XVERGLEICH ist:

XVERGLEICH(lookup_value, lookup_array, [match_mode], [search_mode])

dabei gilt

lookup_value ist ein Wert, nach dem es gesucht werden muss.

lookup_array ist ein Array oder Bereich, in dem es gesucht werden soll.

match_mode ist ein optionales Argument. Folgende Werte stehen zur Verfügung:

  • 0 (standardmäßig festgelegt) gibt die genaue Übereinstimmung zurück; Wenn es keine Übereinstimmung gibt, wird stattdessen „N/A“ zurückgegeben.
  • -1 gibt die genaue Übereinstimmung zurück; ist keine vorhanden, wird der nächstkleinere Artikel zurückgegeben.
  • 1 gibt die genaue Übereinstimmung zurück; Ist keine vorhanden, wird der nächstgrößere Artikel zurückgegeben.
  • 2 ist ein Wildcard-Match.

search_mode ist ein optionales Argument. Folgende Werte stehen zur Verfügung:

  • 1 startet eine Suche beim ersten Element (standardmäßig festgelegt).
  • -1 startet eine Rückwärtssuche, also beim letzten Eintrag.
  • 2startet eine binäre Suche, wobei das lookup_array in aufsteigender Reihenfolge sortiert wird. Bei Nichtsortierung werden ungültige Ergebnisse zurückgegeben.
  • -2 startet eine binäre Suche, wobei das lookup_array in absteigender Reihenfolge sortiert wird. Bei Nichtsortierung werden ungültige Ergebnisse zurückgegeben.

Zu den Platzhalterzeichen gehören das Fragezeichen (?), das einem einzelnen Zeichen entspricht, und das Sternchen (*), das mehreren Zeichen entspricht. Wenn Sie ein Fragezeichen oder ein Sternchen suchen möchten, geben Sie vor dem Zeichen eine Tilde (~) ein.

Anwendung der Funktion XVERGLEICH:

  1. Wählen Sie die Zelle aus, in der Sie das Ergebnis anzeigen möchten.
  2. Klicken Sie auf das Symbol Funktion
    in der oberen Symbolleiste
    oder klicken Sie mit der rechten Maustaste in eine ausgewählte Zelle und wählen Sie die Option Funktion einfügen aus dem Menü
    oder klicken Sie auf das Symbol
    in der Bearbeitungsleiste.
  3. Wählen Sie aus der Liste die Funktionsgruppe Suchen und Bezüge.
  4. Klicken Sie auf die Funktion XVERGLEICH.
  5. Geben Sie die erforderlichen Argumente ein, indem Sie sie durch Kommas trennen.
  6. Drücken Sie die Eingabe-Taste.

Das Ergebnis wird in der ausgewählten Zelle angezeigt.
