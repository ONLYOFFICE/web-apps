Funktion REGEXEXTRAHIEREN

Die Funktion REGEXEXTRAHIEREN ist eine der Text- und Datenfunktionen. Sie wird verwendet, um Text aus einer Zeichenfolge basierend auf einem angegebenen Muster eines regulären Ausdrucks zu extrahieren. Die Funktion kann die erste Übereinstimmung, alle Übereinstimmungen oder Erfassungsgruppen aus der ersten Übereinstimmung zurückgeben.

Syntax

REGEXEXTRAHIEREN(text; muster; [rückgabe_modus]; [beachten_groß_klein])

Die Funktion REGEXEXTRAHIEREN hat folgende Argumente:

Argument Beschreibung
text Die Textzeichenfolge oder der Verweis auf eine Zelle, die den Text enthält, aus dem Sie übereinstimmende Teilzeichenfolgen extrahieren möchten.
muster Der reguläre Ausdruck („Regex"), der das Textmuster beschreibt, das Sie extrahieren möchten.
rückgabe_modus Eine Zahl, die angibt, welche Zeichenfolgen extrahiert werden sollen. Dies ist ein optionales Argument. Wenn es weggelassen wird, ist der Standardwert 0. Die möglichen Werte sind in der folgenden Tabelle aufgeführt.
beachten_groß_klein Legt fest, ob beim Abgleich die Groß-/Kleinschreibung beachtet wird. Dies ist ein optionales Argument. Wenn es weggelassen wird, wird die Groß-/Kleinschreibung beachtet (0). Geben Sie 0 für einen Abgleich mit Berücksichtigung der Groß-/Kleinschreibung oder 1 für einen Abgleich ohne Berücksichtigung der Groß-/Kleinschreibung ein.

Das Argument rückgabe_modus kann einen der folgenden Werte annehmen:

Wert Beschreibung
0 Standardmäßig festgelegt. Gibt die erste Zeichenfolge zurück, die dem Muster entspricht.
1 Gibt alle Zeichenfolgen, die dem Muster entsprechen, als Array zurück.
2 Gibt Erfassungsgruppen aus der ersten Übereinstimmung als Array zurück.

Hinweise

Beim Schreiben von Regex-Mustern können Symbole, sogenannte „Tokens", verwendet werden, die mit einer Vielzahl von Zeichen übereinstimmen. Im Folgenden finden Sie einige einfache Tokens zur Referenz:

  • [0-9]: eine beliebige Ziffer
  • [a-z]: ein Zeichen im Bereich von a bis z
  • [A-Z]: ein Zeichen im Bereich von A bis Z
  • .: ein beliebiges einzelnes Zeichen
  • *: null oder mehr des vorangehenden Zeichens
  • +: ein oder mehr des vorangehenden Zeichens
  • (): Erfassungsgruppe – wird verwendet, um Teile der Übereinstimmung zu erfassen und zurückzugeben

Erfassungsgruppen sind Teile eines Regex-Musters, die von Klammern „(...)" umgeben sind. Sie ermöglichen es, einzelne Teile einer Übereinstimmung separat zurückzugeben.

Die Funktion REGEXEXTRAHIEREN gibt immer Textwerte zurück. Sie können numerische Ergebnisse mithilfe der Funktion WERT wieder in Zahlen umwandeln.

Wenn keine Übereinstimmung gefunden wird, gibt die Funktion einen #NV-Fehler zurück.
