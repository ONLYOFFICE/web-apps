MATRIXZUTEXT-Funktion

Die MATRIXZUTEXT-Funktion ist eine der Text und Datenfunktionen. Sie wird verwendet, um eine Textdarstellung einer Matrix zurückzugeben.

Die Formelsyntax der Funktion MATRIXZUTEXT ist:

=MATRIXZUTEXT(Matrix, [Format])

dabei gilt

Matrix ist der Datenbereich, der als Text zurückgegeben werden soll.

[Format] ist ein optionales Argument. Folgende Werte stehen zur Verfügung:

  • 0, die Werte in der Textzeichenfolge werden durch Komma getrennt (standardmäßig festgelegt).
  • 1, um jeden Textwert in doppelte Anführungszeichen zu setzen (außer Zahlen, Wahr/Falsch-Werte und Fehler), ein Semikolon als Trennzeichen zu verwenden und die gesamte Textzeichenfolge in geschweifte Klammern zu setzen.

Anwendung der Funktion MATRIXZUTEXT:

  1. Wählen Sie die Zelle aus, in der Sie das Ergebnis anzeigen möchten.
  2. Klicken Sie auf das Symbol Funktion
    in der oberen Symbolleiste
    oder klicken Sie mit der rechten Maustaste in eine ausgewählte Zelle und wählen Sie die Option Funktion einfügen aus dem Menü
    oder klicken Sie auf das Symbol
    in der Bearbeitungsleiste.
  3. Wählen Sie aus der Liste die Funktionsgruppe Text und Daten.
  4. Klicken Sie auf die Funktion MATRIXZUTEXT.
  5. Geben Sie die erforderlichen Argumente ein, indem Sie sie durch Kommas trennen.
  6. Drücken Sie die Eingabe-Taste.

Das Ergebnis wird in der ausgewählten Zelle angezeigt.
