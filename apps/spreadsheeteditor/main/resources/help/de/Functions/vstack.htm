VSTAPELN-Funktion

Die VSTAPELN-Funktion ist eine der Suchen- und Bezügefunktionen. Sie wird verwendet, um Arrays vertikal in einem Array zu stapeln.

Bitte beachten Sie, dass dies eine Matrixformel ist. Um mehr zu erfahren, lesen Sie bitte den Artikel Matrixformeln einfügen.

Die Formelsyntax der Funktion VSTAPELN ist:

VSTAPELN (array1, [array2], ...)

dabei gilt

array wird verwendet, um die anzuhängenden Arrays festzulegen.

Anwendung der Funktion VSTAPELN:

  1. Wählen Sie die Zelle aus, in der Sie das Ergebnis anzeigen möchten,
  2. Klicken Sie auf das Symbol Funktion
    in der oberen Symbolleiste
    oder klicken Sie mit der rechten Maustaste in eine ausgewählte Zelle und wählen Sie die Option Funktion einfügen aus dem Menü
    oder klicken Sie auf das Symbol
    in der Bearbeitungsleiste.
  3. Wählen Sie aus der Liste die Funktionsgruppe Suchen und Bezüge.
  4. Klicken Sie auf die Funktion VSTAPELN.
  5. Geben Sie die erforderlichen Argumente ein, indem Sie sie durch Kommas trennen.
  6. Drücken Sie die Eingabe-Taste.

Das Ergebnis wird in der ausgewählten Zelle angezeigt.
