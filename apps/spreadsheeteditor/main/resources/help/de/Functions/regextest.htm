Funktion REGEXTESTEN

Die Funktion REGEXTESTEN ist eine der Text- und Datenfunktionen. Sie wird verwendet, um zu prüfen, ob ein Teil des angegebenen Textes einem Muster eines regulären Ausdrucks entspricht, und gibt WAHR oder FALSCH zurück.

Syntax

REGEXTESTEN(text; muster; [beachten_groß_klein])

Die Funktion REGEXTESTEN hat folgende Argumente:

Argument Beschreibung
text Die Textzeichenfolge oder der Verweis auf eine Zelle, die den Text enthält, den Sie anhand des Musters testen möchten.
muster Der reguläre Ausdruck („Regex"), der das Textmuster beschreibt, das Sie abgleichen möchten.
beachten_groß_klein Legt fest, ob beim Abgleich die Groß-/Kleinschreibung beachtet wird. Dies ist ein optionales Argument. Wenn es weggelassen wird, wird die Groß-/Kleinschreibung beachtet (0). Geben Sie 0 für einen Abgleich mit Berücksichtigung der Groß-/Kleinschreibung oder 1 für einen Abgleich ohne Berücksichtigung der Groß-/Kleinschreibung ein.

Hinweise

Beim Schreiben von Regex-Mustern können Symbole, sogenannte „Tokens", verwendet werden, die mit einer Vielzahl von Zeichen übereinstimmen. Im Folgenden finden Sie einige einfache Tokens zur Referenz:

  • [0-9]: eine beliebige Ziffer
  • [a-z]: ein Zeichen im Bereich von a bis z
  • [A-Z]: ein Zeichen im Bereich von A bis Z
  • .: ein beliebiges einzelnes Zeichen
  • *: null oder mehr des vorangehenden Zeichens
  • +: ein oder mehr des vorangehenden Zeichens
  • ^: Anfang der Zeichenfolge
  • $: Ende der Zeichenfolge
  • \: Escapezeichen (z. B. entspricht \( einer wörtlichen Klammer)
  • {n}: genau n Vorkommen des vorangehenden Zeichens

Die Funktion gibt WAHR zurück, wenn das Muster mit einem Teil des Textes übereinstimmt, und FALSCH, wenn keine Übereinstimmung gefunden wird.
