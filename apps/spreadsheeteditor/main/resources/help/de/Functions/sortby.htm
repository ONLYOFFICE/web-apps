SORTIERENNACH-Funktion

Die SORTIERENNACH-Funktion ist eine der Suchen- und Bezügefunktionen. Sie wird verwendet, um einen Datenbereich oder ein Array basierend auf den Werten in einem entsprechenden Bereich oder Array zu sortieren.

Formelsyntax

SORTIERENNACH(Matrix, nach_Matrix, [Sortierreihenfolge], ...)

Die SORTIERENNACH-Funktion hat die folgenden Argumente:

Argument Beschreibung
Matrix Eine Reihe von Daten zum Sortieren.
nach_Matrix Ein Array oder Bereich, der zum Sortieren verwendet wird.
Sortierreihenfolge Ein optionales Argument. Eine Zahl, die die Sortierreihenfolge bestimmt. Die möglichen Werte sind in der folgenden Tabelle aufgeführt.

Das Argument Sortierreihenfolge kann eines der folgenden sein:

Zahlenwert Bedeutung
1 oder weggelassen Aufsteigende Reihenfolge (standardmäßig)
-1 Absteigende Reihenfolge

Notizen

Bitte beachten Sie, dass dies eine Matrixformel ist. Um mehr zu erfahren, lesen Sie bitte den Artikel Matrixformeln einfügen.
