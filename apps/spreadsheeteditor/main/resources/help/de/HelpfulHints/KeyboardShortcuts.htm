Tastenkombinationen für Key-Tipps

Verwenden Sie Tastenkombinationen für einen schnelleren und einfacheren Zugriff auf die Funktionen des Tabellenkalkulationseditors ohne eine Maus zu verwenden.

  1. Drücken Sie die Alt-Taste (Option für macOS), um alle wichtigen Tipps für die Kopfzeile des Editors, die obere Symbolleiste, die rechte und linke Seitenleiste und die Statusleiste einzuschalten.
  2. Drücken Sie den Buchstaben, der dem Element entspricht, das Sie verwenden möchten. Die zusätzlichen Tastentipps können je nach gedrückter Taste angezeigt werden. Die ersten Tastentipps werden ausgeblendet, wenn zusätzliche Tastentipps angezeigt werden.

    Um beispielsweise auf die Registerkarte Einfügen zuzugreifen, drücken Sie Alt (Option für macOS), um alle Tipps zu den Primärtasten anzuzeigen.

    Drücken Sie den Buchstaben I, um auf die Registerkarte Einfügen zuzugreifen, und Sie sehen alle verfügbaren Verknüpfungen für diese Registerkarte.

    Drücken Sie dann den Buchstaben, der dem zu konfigurierenden Element entspricht.

  3. Drücken Sie Alt (Option für macOS), um alle Tastentipps auszublenden, oder drücken Sie ESC, um zur vorherigen Gruppe von Tastentipps zurückzukehren.

In der folgenden Liste finden Sie die gängigsten Tastenkombinationen.

Bitte beachten Sie: Bei macOS enthalten einige Tastenkombinationen Tasten wie POS1, Ende, Bild auf und Bild ab, die nur auf der erweiterten Tastatur verfügbar sind. Wenn Sie diese Tasten nicht haben, verwenden Sie die oben angegebenen Tastenkombinationen (d. h. verwenden Sie ^ STRG/⌘ Cmd+← oder Fn+← anstelle von POS1, ^ STRG/⌘ Cmd+→ oder Fn+→ anstelle von Ende, Fn+↑ anstelle von Bild auf, Fn+↓ anstelle von Bild ab).
