Wort durch Synonym ersetzen

Wenn Sie dasselbe Wort mehrmals verwenden oder ein Wort nicht ganz das richtige Wort ist, können Sie im ONLYOFFICE Tabelleneditor nach Synonymen suchen. Die Antonyme werden auch angezeigt.

Ab ONLYOFFICE Docs 8.2 sind standardmäßig keine Plugins mehr in den Editoren enthalten. Die Plugins müssen über die Plugin-Verwaltung installiert werden.

  1. Wählen Sie das Wort in Ihrer Tabelle aus.
  2. Öffnen Sie die Registerkarte Plugins und wählen Sie den Menüpunkt
    Thesaurus aus.
  3. Die Synonyme und Antonyme werden in der linken Seitenleiste angezeigt.
  4. Klicken Sie auf ein Wort, um das Wort in Ihrer Tabelle zu ersetzen.
