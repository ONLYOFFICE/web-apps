Zellen externe Links hinzufügen

In der Tabellenkalkulation können Sie einen externen Link zu einer Zelle oder einem Zellbereich in einer anderen Arbeitsmappe erstellen. Die externen Links zu Zellen können zwischen Dateien innerhalb des aktuellen Portals (im Online-Editor) oder zwischen lokalen Dateien (im Desktop-Editor) erstellt werden.

Wenn sich Daten in der Quellarbeitsmappe ändern, können Sie Werte in der Zielarbeitsmappe aktualisieren, ohne sie erneut manuell zu kopieren.

Um einen externen Link hinzuzufügen,

  1. öffnen Sie die Quellarbeitsmappe und die Zielarbeitsmappe,
  2. kopieren Sie in der Quellarbeitsmappe eine Zelle oder einen Zellbereich (Strg+C),
  3. fügen Sie in der Zielarbeitsmappe die kopierten Daten ein (Strg+V),
  4. klicken Sie auf die Schaltfläche Spezielles Einfügen und wählen Sie die Option Hyperlink einfügen (Strg+N).

Der Link wird hinzugefügt. Wenn Sie auf die Zelle klicken, die einen externen Link enthält, sieht dies in der Formelleiste wie folgt aus: ='[SourceWorkbook.xlsx]Sheet1'!A1.

Um die hinzugefügten externen Links zu aktualisieren,

  1. wechseln Sie zur Registerkarte Daten,
  2. klicken Sie auf die Schaltfläche Externe Links,
  3. wählen Sie den erforderlichen Link in der Liste aus,

  4. Klicken Sie auf die Schaltfläche Werte aktualisieren oder auf den Pfeil daneben und wählen Sie aus, ob Sie Werte aktualisieren (die derzeit ausgewählt werden) oder Alles aktualisieren möchten.

Der Status wird auf „OK“ geändert.

Wenn Sie die Arbeitsmappe öffnen, die externe Links enthält, wird die Warnung angezeigt:

Klicken Sie auf Aktualisieren, um alle externen Links zu aktualisieren.

Um die hinzugefügten externen Links zu unterbrechen,

  1. wechseln Sie zur Registerkarte Daten,
  2. klicken Sie auf die Schaltfläche Externe Links,
  3. wählen Sie den erforderlichen Link in der Liste aus,
  4. Um die hinzugefügten externen Links zu unterbrechen, klicken Sie auf die Schaltfläche Verknüpfungen löschen oder klicken Sie auf den Pfeil daneben und wählen Sie aus, ob Sie Verknüpfungen löschen (die derzeit ausgewählt werden) oder Alle Verknüpfungen aufheben möchten.

Die Verknüpfungen werden unterbrochen und die Werte werden nicht aktualisiert.
