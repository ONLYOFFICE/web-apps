Zielwertsuche

ONLYOFFICE Tabellenkalkulationseditor bietet eine Zielwertsuche-Funktion, mit der Sie bestimmen können, welche Zahlen in die Formel eingesetzt werden sollen, um das gewünschte und bereits bekannte Ergebnis zu erhalten.

  1. Öffnen Sie die Registerkarte Daten und klicken Sie in der oberen Symbolleiste auf das Symbol Zielwertsuche.
  2. Wählen Sie im geöffneten Fenster Zielwertsuche die erforderlichen Optionen aus:

    • In der Zelle einstellen - Geben Sie den Verweis auf die Zelle ein, die eine Formel enthält. Sie können das Symbol
      verwenden, um die Zelle auszuwählen.
    • Wert - Geben Sie das gewünschte Ergebnis in die Zelle mit der Formel ein.
    • Durch Veränderung der Zelle - Geben Sie den Verweis auf die Zelle ein, die den Wert enthält, den Sie ändern möchten. Sie können das Symbol
      verwenden, um die Zelle auszuwählen. Auf diese Zelle muss durch die in der Zelle befindliche Formel verwiesen werden, die Sie im Feld Zelle festlegen angegeben haben.
  3. Klicken Sie auf OK.
  4. Das Fenster Status der Parameterauswahl zeigt das Ergebnis an.

  5. Klicken Sie auf OK, um die Werte in den Zellen zu ersetzen, die in den Feldern In der Zelle einstellen und Durch Veränderung der Zelle angegeben sind.
