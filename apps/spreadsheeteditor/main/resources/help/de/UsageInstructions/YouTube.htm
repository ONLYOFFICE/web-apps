Video einfügen

Sie können ein Video in der Tabellenkalkulation in Ihre Tabelle einfügen. Es wird als Bild angezeigt. Durch Doppelklick auf das Bild wird der Videodialog geöffnet. Hier können Sie das Video starten.

Ab ONLYOFFICE Docs 8.2 sind standardmäßig keine Plugins mehr in den Editoren enthalten. Die Plugins müssen über die Plugin-Verwaltung installiert werden.

  1. Kopieren Sie die URL des Videos, das Sie einbetten möchten.
    (die vollständige Adresse wird in der Adresszeile Ihres Browsers angezeigt)
  2. Gehen Sie zu Ihrer Tabelle und platzieren Sie den Cursor an der Stelle, an der Sie das Video einfügen möchten.
  3. Öffnen Sie die Registerkarte Plugins und wählen Sie den Menüpunkt
    YouTube aus.
  4. Fügen Sie die URL ein und klicken Sie auf OK.
  5. Überprüfen Sie, ob Sie das richtige Video eingefügt haben, und klicken Sie auf die Schaltfläche OK unter dem Video.

Das Video ist jetzt in Ihrer Tabelle eingefügt.
