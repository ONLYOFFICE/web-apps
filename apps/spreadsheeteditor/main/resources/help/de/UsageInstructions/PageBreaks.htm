Seitenumbrüche einfügen

In der Tabellenkalkulation können Sie einen Seitenumbruch hinzufügen, um Ihre Tabelle für den Druck auf separaten Seiten vorzubereiten.

Um an der aktuellen Cursorposition einen Seitenumbruch einzufügen:

  1. Klicken Sie auf den Pfeil neben dem Symbol Umbrüche auf der Registerkarte Layout der oberen Symbolleiste.
  2. Wählen Sie im Menü die Option Seitenumbruch einfügen.

Die hinzugefügten Seitenumbrüche werden in Ihrer Tabelle durch gepunktete Linien angezeigt.

Seitenumbrüche hängen von den eingestellten Ränder, Orientierung und Größe ab. Wenn Sie die Parameter ändern, wiederholen Sie die Schritte 2 und 3 der obigen Anweisungen.

Um die Pausen auf die Standardwerte zurückzusetzen:

  1. Klicken Sie auf den Pfeil neben dem Symbol Umbrüche auf der Registerkarte Layout der oberen Symbolleiste.
  2. Wählen Sie die Option Alle Seitenumbrüche zurücksetzen.
  3. Die Standardeinstellungen sind wie folgt:
    • Ränder - Normal;
    • Größe - A4;
    • Orientierung - Hochformat.

Um einen Seitenumbruch zu entfernen:

  1. Klicken Sie auf den Pfeil neben dem Symbol Umbrüche auf der Registerkarte Layout der oberen Symbolleiste.
  2. Wählen Sie die Option Seitenumbruch entfernen.
