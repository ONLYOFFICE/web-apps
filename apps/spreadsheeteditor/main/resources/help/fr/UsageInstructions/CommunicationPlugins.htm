Communiquer lors de l'édition

Dans l'éditeur de classeurs ONLYOFFICE, vous pouvez toujours rester en contact avec vos collègues et utiliser des messageries en ligne populaires, telles que Telegram et Rainbow.

Les module complémentaires Telegram et Rainbow ne sont pas installés par défaut. Pour en savoir plus sur l'installation, veuillez consulter le guide approprié: Ajouter des modules complémentaires à l'éditeur de bureau ONLYOFFICE Ajouter des modules complémentaires au cloud ONLYOFFICE ou Ajouter de nouveaux modules complémentaires aux éditeurs de serveur ou installer les modules complémentaires à l'aide de Gestionnaire de plugins.

Bot

Pour connecter ONLYOFFICE à Telegram,

  1. Utilisez le lien d'installation et cliquer sur le bouton COMMENCER.

    La prochaine fois que vous démariez le bot, utilisez la commande /start pour accéder au menu de démarrage et les options disponibles.

  2. Pour créer et modifier des fichiers:

    • Cliquez sur le bouton Créer dans le menu.
    • Sélectionnez le format de fichier nécessaire: Document, Classeur ou Présentation.
    • Saisissez le nom du fichier.
    • Le bot enverra un message contenant un lien vers le fichier créé. Cliquez sur le lien pour ouvrir le fichier dans l'éditeur ONLYOFFICE correspondant (selon le type de fichier choisi).
    • Une fois la session d'édition terminée, le bot enverra le fichier avec les modifications enregistrées.

      Le lien d'édition sera désactivé une fois le fichier modifié envoyé. Pour rouvrir le fichier, répondez simplement au message contenant le fichier avec n'importe quel texte.

  3. Pour collaborer sur des fichiers :
    • Pour collaborer sur le fichier créé, transférez le message contenant le lien à d'autres utilisateurs Telegram.
    • Le lien reste actif pendant 24 heures. Chaque fois on fait on clic dessus durant cette période, le minuteur se réinitialise pour 24 heures supplémentaires.
    • Si le lien a expiré, un message d'erreur s'affichera. Une fois que tous les participants ayant accédé au lien ont fermé l'éditeur, le fichier mis à jour (si des modifications ont été apportées) sera instantanément envoyé à toutes les personnes concernées.
  4. Pour ouvrir un fichier local:
    • Envoyez un fichier dans le chat sans sélectionner de commande.
    • Sélectionnez les options suivantes dans le menu du bot: /start → Ouvrir → Télécharger un fichier depuis l'appareil.

      Dans les deux cas, le bot enverra un message avec le lien d'ouverture.

      Si vous envoyez plusieurs fichiers dans un même message, le lien sera généré pour le premier fichier du message.

  5. Pour convertir un fichier :
    • Sélectionnez les options suivantes dans le menu du bot: /start → Convertir.
    • Envoyez le fichier depuis votre appareil au bot.
    • Choisissez le format vers lequel vous souhaitez convertir le fichier téléchargé. Consultez la liste des formats pris en charge sur la page GitHub.

      Une fois terminé, le bot enverra le fichier converti directement dans le chat dans le format sélectionné.

  6. Pour obtenir des conseils sur l'utilisation du bot, utilisez la commande/help.

Utiliser le bot Telegram dans les discussions de groupe

Les fonctionnalités du bot lors de l'utilisation dans un groupe diffèrent de celles en conversation individuelle:

  • La création de fichiers est réalisée via trois commandes : /document, /feuille de calcul et /présentation. Les utilisateurs ne définissent pas manuellement le nom du fichier. Il est automatiquement généré selon le modèle suivant: Nouveau [format] Date Heure UTC.
  • L'interaction avec le bot varie selon les permissions d'administration. Sans permissions d'administrateur, vous devez répondre directement au bot, qui ne peut pas accéder aux messages de fichiers transférés (sauf les siens). Avec les permissions d'administrateur, le bot voit tous les messages, ce qui vous permet de transférer des fichiers sans répondre directement.
  • Pour générer un nouveau lien d'édition pour un fichier, répondez au message contenant le fichier et mentionnez le bot @ONLYOFFICE_bot. Dans certains cas, le bot peut nécessiter des permissions d'accès administrateur dans le groupe.
  • Lors de l'ouverture de fichiers, le bot ne répondra pas aux fichiers simplement envoyés dans le chat. Pour ouvrir un fichier, utilisez la commande /open et répondez au bot avec le fichier joint dans un message de réponse.
  • La langue du bot ne peut pas être définie de manière universelle pour le groupe. Il répondra plutôt dans la langue choisie par chaque utilisateur individuellement.

    Par défaut, le bot ONLYOFFICE utilise la langue du profil Telegram de l'utilisateur. Si cette langue n'est pas disponible dans les traductions du bot, la langue anglaise est utilisée. Si vous souhaitez changer manuellement la langue du bot, entrez la commande /lang.

  • La conversion de fichiers n'est pas disponible.

Rainbow

Pour démarrer une nouvelle conversation sur Rainbow à l'aide du module complémentaire,

  1. Passez à l'onglet Modules complémentaires et cliquez sur
    Rainbow,
  2. enregistrez le nouveau compte en cliquant sur le bouton Sign up (S'inscrire) ou connectez-vous à votre compte existant. Pour ce faire, saisissez votre adresse e-mail dans le champ approprié et cliquez sur Continue (Continuer),

  3. ensuite saisissez votre mot de passe pour ce compte,
  4. cochez la case Keep my session alive (Garder ma session active) si vous souhaitez sauvegarder les informations d'identification pour la session en cours et cliquez sur le bouton Connect (Connecter).
  5. remplissez les champs dans la fenêtre Your identity (Votre identifiants).

Maintenant vous pouvez démarrer une conversation dans Rainbow et travailler à partir des éditeurs ONLYOFFICE.
