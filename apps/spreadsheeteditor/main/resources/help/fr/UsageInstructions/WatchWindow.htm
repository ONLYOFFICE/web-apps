Fenêtre Espions

Lorsque l'on travaille avec de grands ensembles de données dans l'éditeur de classeurs, la Fenêtre Espions peut être utile pour surveiller les cellules et les formules. La Fenêtre Espions permet de surveiller les modifications apportées aux cellules qui ne sont pas visibles dans une feuille de calcul. Vous pouvez basculer vite vers la cellule nécessaire en double-cliquant sur celle-ci dans la Fenêtre Espions.

La Fenêtre Espions permet de suivre les éléments suivants: Classeur, Feuille de calcul, Nom, Cellule, Valeur et Formule.

Pour ajouter un nouveau espion,

  1. Passez à l'onglet Formule.
  2. Cliquez sur le bouton Fenêtre Espions.
  3. Cliquez sur Ajouter un espion dans la Fenêtre Espions.
  4. Sélectionnez la plage de données à surveiller.
  5. Cliquez sur Fermer pour revenir à la feuille de calcul ou continuez à travailler sur la feuille de calcul en gardant la fenêtre ouverte. Toutes modifications apportées seront affichées dans la fenêtre espion.

Pour supprimer un espion,

  1. Passez à l'onglet Formule.
  2. Cliquez sur le bouton Fenêtre Espions.
  3. Sélectionnez l'espion approprié ou plusieurs espions en utilisant le raccourci clavier Ctrl+bouton gauche de la souris.
  4. Cliquez sur le bouton Supprimer l'espion et sélectionnez Supprimer l'espion ou Supprimer tout.
