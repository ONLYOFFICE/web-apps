Fonction REGEX.EXTRAIRE

La fonction REGEX.EXTRAIRE est l'une des fonctions de texte et de données. Cette fonction vous permet d'extraire du texte d'une chaîne en fonction d'une expression régulière fournie. La fonction peut renvoyer la première correspondance, toutes les correspondances ou captures de groupes à partir de la première correspondance.

Syntaxe

REGEX.EXTRAIRE(texte, modèle, [mode_renvoi], [respect_casse])

Les arguments de la fonction REGEX.EXTRAIRE sont les suivants:

Argument Description
texte Texte ou référence à une cellule contenant le texte à partir duquel vous souhaitez extraire des chaînes correspondantes.
modèle Expression régulière ("regex") qui décrit le modèle de texte que vous souhaitez extraire.
mode_renvoi Nombre qui spécifie les chaînes que vous souhaitez extraire. es valeurs possibles sont les suivantes: C'est un argument facultatif. Par défaut, le mode de renvoi est 0. Les valeurs possibles sont énumérées dans le tableau ci-dessous.
respect_casse Définit si la correspondance respecte la casse. C'est un argument facultatif. Si cet argument est omis, la correspondance respecte la casse (0). Saisissez 0 pour la correspondance respectant la casse ou 1 pour la correspondance non sensible à la casse.

Les arguments de mode_renvoi sont les suivants:

Valeur Description
0 La valeur par défaut Renvoyer la première chaîne qui correspond au modèle.
1 Renvoyer toutes les chaînes qui correspondent au modèle sous forme de tableau.
2 Renvoyer des groupes de capture à partir de la première correspondance sous forme de tableau

Remarques

Lors de l'écriture de modèles d'expression régulière, vous pouvez utiliser des symboles appelés jetons qui correspondent à une variété de caractères. Voici quelques jetons simples à référencer:

  • [0-9]: n'importe quel chiffre numérique
  • [a-z]:: caractère dans la plage de a à z
  • [A-Z]: caractère dans la plage de A à Z
  • .: n'importe quel caractère
  • *: zéro ou plusieurs fois le caractère précédent
  • +: une ou plusieurs fois le caractère précédent
  • (): groupe de capture - pour capturer et renvoyer les fragments de la correspondance.

Les groupes de capture sont des parties d'une expression régulière délimitées par des parenthèses "(...)". Vous pouvez renvoyer fragment distincts d'une seule correspondance chaque à part.

La fonction REGEX.EXTRAIRE revoie toujours des valeurs de texte. Vous pouvez convertir ces résultats en nombre à l'aide de la fonction VALEUR.

Si aucune correspondance n'est trouvée, la fonction renvoie la valeur d'erreur #N/A.
