Fonction REGEX.REMPLACER

La fonction REGEX.REMPLACER est l'une des fonctions de texte et de données. Cette fonction vous permet de remplacer le texte d'une chaîne par une autre chaîne, en fonction d'une expression régulière fournie.

Syntaxe

REGEX.REMPLACER(texte, modèle, remplacement, [occurrence], [respect_casse])

Les arguments de la fonction REGEX.REMPLACER sont les suivants:

Argument Description
texte Texte ou référence à une cellule contenant le texte dans lequel vous souhaitez remplacer des chaînes.
modèle Expression régulière ("regex") qui décrit le modèle de texte à remplacer.
remplacement Texte que vous souhaitez remplacer dans la correspondance modèle. Vous pouvez utiliser $1, $2 pour faire référence aux groupes de capture qui font partie d'un modèle.
occurrence Définit l'occurrence du modèle que vous voulez remplacer. C'est un argument facultatif. Si cet argument est omis, la valeur par défaut est 0, ce qui remplace toutes les occurrences. Un nombre positif remplace cette occurrence spécifique. Un nombre négatif remplace les occurrences en effectuant une recherche à partir de la fin.
respect_casse Définit si la correspondance respecte la casse. C'est un argument facultatif. Si cet argument est omis, la correspondance respecte la casse (0). Saisissez 0 pour la correspondance respectant la casse ou 1 pour la correspondance non sensible à la casse.

Remarques

Lors de l'écriture de modèles d'expression régulière, vous pouvez utiliser des symboles appelés jetons qui correspondent à une variété de caractères. Voici quelques jetons simples à référencer:

  • [0-9]: n'importe quel chiffre numérique
  • [a-z]:: caractère dans la plage de a à z
  • [A-Z]: caractère dans la plage de A à Z
  • .: n'importe quel caractère
  • *: zéro ou plusieurs fois le caractère précédent
  • +: une ou plusieurs fois le caractère précédent
  • (): groupe de capture - le texte capturé peut être référencé dans le remplacement par $1, $2, etc.
  • {n}: exactement n fois le caractère précédent
  • \: caractère d'échappement pour les caractères spéciaux.

Les groupes de capture sont des parties d'une expression régulière délimitées par des parenthèses "(...)". L'argument de remplacement peut référencer les groupes capturés en utilisant :

  • $1 - premier groupe de capture
  • $2 - deuxième groupe de capture
  • $n - le nième groupe de capture.

La fonction renvoie la chaîne de texte modifiée. Le texte d'origine dans la cellule source n'est pas modifié.
