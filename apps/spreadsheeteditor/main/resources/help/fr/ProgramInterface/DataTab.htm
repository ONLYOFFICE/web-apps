Onglet Données

L'onglet Données permet de gérer les données dans l'éditeur de classeurs.

L'onglet Données dans l'éditeur de classeurs en ligne:

L'onglet Données dans l'éditeur de classeurs de bureau:

En utilisant cet onglet, vous pouvez:

  • trier et filtrer les données,
  • transformer texte en colonnes,
  • supprimer les valeurs en double de la plage de données,
  • grouper et dissocier des données,
  • régler les paramètres de validation de données,
  • obtenir les données à partir d'un fichier TXT/CSV,
  • afficher d'autres fichiers auxquels le classer est lié à l'aide du bouton Lien externe.
