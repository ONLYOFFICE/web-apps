Função SORTBY

A função SORTBY é uma das funções de pesquisa e referência. É usado para classificar um intervalo de dados ou uma matriz com base nos valores de um intervalo ou matriz correspondente.

Sintaxe

SORTBY(matriz, por_matriz, [ordem_de_classificação], ...)

A função SORTBY tem os seguintes argumentos:

Argumento Descrição
matriz Um intervalo de dados para classificar.
por_matriz Uma matriz ou intervalo usado para classificação.
ordem_de_classificação Um argumento opcional. Um número que determina a ordem de classificação. Os valores possíveis estão listados na tabela abaixo.

O argumento ordem_de_classificação pode ser um dos seguintes:

Valor numérico Significado
1 ou omitido Ordem ascendente (por padrão)
-1 Ordem decrescente

Notas

Observe que essa é uma fórmula de matriz. Para saber mais, leia o artigo Inserir fórmulas de matriz.
