Função IMPORTRANGE

A função IMPORTRANGE é uma das funções de texto e dados. É usado para importar um intervalo de células de uma planilha especificada.

Sintaxe

IMPORTRANGE("planilha_url", "string_de_intervalo")

A função IMPORTRANGE tem os seguintes argumentos:

Argumento Descrição
planilha_url É usado para especificar o URL da planilha da qual os dados são importados.
string_de_intervalo Uma cadeia de caracteres que especifica o intervalo a ser importado.
