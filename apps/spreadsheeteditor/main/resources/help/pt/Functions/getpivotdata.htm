Função GETPIVOTDATA

A função GETPIVOTDATA é uma das funções de pesquisa e referência. É usado para extrair dados armazenados em uma tabela dinâmica.

Sintaxe

GETPIVOTDATA(campo_de_dados, tabela_dinâmica, [campo], [item], …)

A função GETPIVOTDATA tem os seguintes argumentos:

Argumento Descrição
campo_de_dados É usado para definir o nome do campo da tabela dinâmica que contém os dados necessários. O argumento precisa estar entre aspas.
tabela_dinâmica É usado para se referir a qualquer célula, intervalo de células ou intervalo nomeado de células em uma tabela dinâmica. O argumento é usado para determinar qual tabela dinâmica contém os dados necessários.
campo, item Um par de campo/item. Há até 126 pares de nomes de campos e nomes de itens que podem ser usados para descrever os dados necessários. Os pares podem estar em qualquer ordem.
