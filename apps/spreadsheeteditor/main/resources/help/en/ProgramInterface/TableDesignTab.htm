Table Design tab

The Table Design tab in the Spreadsheet Editor allows editing formatted tables.

Starting from ONLYOFFICE Docs version 9.1, the Table Design tab is hidden by default and will open only while working with tables.

The corresponding window of the Desktop Spreadsheet Editor:

Using this tab, you can:

  • resize tables,
  • manage rows and columns and their display,
  • remove duplicates,
  • convert to range,
  • insert slicers,
  • create pivot tables,
  • add alternative title and description,
  • choose one of the predefined table styles.
