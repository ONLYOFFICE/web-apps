Insert page breaks

In the Spreadsheet Editor, you can add a page break to get your spreadsheet ready for printing in separate pages.

To insert a page break at the current cursor position:

  1. click the arrow next to the Breaks icon on the Layout tab of the top toolbar,
  2. select the Insert page break option from the menu.

The added page breaks are indicated in your spreadsheet with dotted lines.

Page breaks depend on the set Margins, Orientation, and Size parameters. If you change the parameters, repeat steps 2 and 3 of the instructions above.

To set the breaks back to default:

  1. click the arrow next to the Breaks icon on the Layout tab of the top toolbar,
  2. select the Reset all page breaks option,
  3. the default settings are as follows:
    • Margin - Normal;
    • Size - A4;
    • Orientation - Portrait.

To remove a page break:

  1. click the arrow next to the Breaks icon on the Layout tab of the top toolbar,
  2. select the Remove page break option.
