Функция СОРТПО

Функция СОРТПО - это одна из поисковых функций. Она возвращает отсортированный диапазон данных или массива на основе значений в соответствующем диапазоне или массиве.

Синтаксис

СОРТПО(массив, по_массиву, [порядок_сортировки], ...)

Функция СОРТПО содержит следующие аргументы:

Аргумент Описание
массив Диапазон ячеек для сортировки.
по_массиву Диапазон ячеек для сортировки по.
порядок_сортировки Это необязательный аргумент. Число, определяющее порядок сортировки. Возможные значения перечислены в таблице ниже.

Аргумент порядок_сортировки может иметь следующие значения:

Числовое значение Значение
1 или не указано По возрастанию (по умолчанию)
-1 По убыванию

Примечания

Обратите внимание, что это формула массива. Чтобы узнать больше, обратитесь к статье Вставка формул массива.
