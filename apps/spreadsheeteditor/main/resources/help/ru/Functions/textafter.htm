Функция ТЕКСТПОСЛЕ

Функция ТЕКСТПОСЛЕ - это одна из функций для работы с текстом и данными. Возвращает текст после символов-разделителей.

Синтаксис функции ТЕКСТПОСЛЕ:

ТЕКСТПОСЛЕ(text,delimiter,[instance_num], [match_mode], [match_end], [if_not_found])

где

text - текст, в котором производится поиск. Использовать подстановочные знаки нельзя. Если text - пустая строка, функция возвращает пустой текст.

delimiter - текст, помечающий точку, после которой нужно извлечь текст.

instance_num - необязательный аргумент. Экземпляр разделителя, после которого нужно извлечь текст. По умолчанию instance_num равен 1. При отрицательном значении поиск начинается с конца.

match_mode - необязательный аргумент. Определяет, учитывается ли регистр в текстовом поиске. По умолчанию регистр учитывается. Используются следующие значения: 0 - с учетом регистра, 1 - без учета регистра.

match_end - необязательный аргумент. Рассматривает конец текста как разделитель. По умолчанию текст является точным совпадением. Используются следующие значения: 0 - не сопоставлять разделитель с концом текста, 1 - сопоставлять разделитель с концом текста.

if_not_found - необязательный аргумент. Задает значение, которое возвращается, если совпадение не найдено.

Чтобы применить функцию ТЕКСТПОСЛЕ,

  1. выделите ячейку, в которой требуется отобразить результат,
  2. щелкните по значку Вставить функцию
    , расположенному на верхней панели инструментов,
    или щелкните правой кнопкой мыши по выделенной ячейке и выберите в меню команду Вставить функцию,
    или щелкните по значку
    перед строкой формул,
  3. выберите из списка группу функций Текст и данные,
  4. щелкните по функции ТЕКСТПОСЛЕ,
  5. введите требуемые аргументы через точку с запятой,
  6. нажмите клавишу Enter.

Результат будет отображен в выбранной ячейке.
