Функция ГСТОЛБИК

Функция ГСТОЛБИК - это одна из поисковых функций. Она используется для того, чтобы горизонтально собрать массивы в один массив.

Обратите внимание, что это формула массива. Чтобы узнать больше, обратитесь к статье Вставка формул массива.

Синтаксис функции ГСТОЛБИК:

ГСТОЛБИК (массив1, [массив2], ...)

где

массив используется, чтобы задать массивы для присоединения.

Чтобы применить функцию ГСТОЛБИК,

  1. выделите ячейку, в которой требуется отобразить результат,
  2. щелкните по значку Вставить функцию
    , расположенному на верхней панели инструментов,
    или щелкните правой кнопкой мыши по выделенной ячейке и выберите в меню команду Вставить функцию,
    или щелкните по значку
    перед строкой формул,
  3. выберите из списка группу функций Поиск и ссылки,
  4. щелкните по функции ГСТОЛБИК,
  5. введите требуемые аргументы через точку с запятой,
  6. нажмите клавишу Enter.

Результат будет отображен в выбранной ячейке.
