Функция ВЗЯТЬ

Функция ВЗЯТЬ - это одна из поисковых функций. Она возвращает строки или столбцы из начала или конца массива.

Обратите внимание, что это формула массива. Чтобы узнать больше, обратитесь к статье Вставка формул массива.

Синтаксис функции ВЗЯТЬ:

ВЗЯТЬ(массив, строки, [столбцы])

где

массив - ссылка на диапазон ячеек, из которого берутся строки или столбцы.

строки - определяет количество строк, которые нужно взять. При отрицательном значении, строки будут браться с конца диапазона.

столбцы - определяет количество столбцов, которые нужно взять. При отрицательном значении, столбцы будут браться с конца диапазона.

Чтобы применить функцию ВЗЯТЬ,

  1. выделите ячейку, в которой требуется отобразить результат,
  2. щелкните по значку Вставить функцию
    расположенному на верхней панели инструментов,
    или щелкните правой кнопкой мыши по выделенной ячейке и выберите в меню команду Вставить функцию,
    или щелкните по значку
    перед строкой формул,
  3. выберите из списка группу функций Поиск и ссылки,
  4. щелкните по функции ВЗЯТЬ,
  5. введите требуемые аргументы через точку с запятой,
  6. нажмите клавишу Enter.

Результат будет отображен в выбранной ячейке.
