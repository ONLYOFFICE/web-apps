Окно контрольных значений

При работе с большими таблицами в редакторе электронных таблиц Окно контрольных значений может пригодиться, чтобы контролировать ячейки и формулы, которые в них используются.

Окно контрольных значений позволяет отслеживать следующие параметры контрольных значений: Книга, Лист, Имя, Ячейка, Значение и Формула.

Чтобы добавить контрольное значение,

  1. Перейдите на вкладку Формула.
  2. Нажмите на кнопку Окно контрольных значений.
  3. Нажмите на кнопку Добавить контрольное значение в Окне контрольных значений.
  4. Выберите нужный диапазон данных для контроля значений.
  5. Нажмите на кнопку Закрыть, чтобы вернуться к рабочему листу.

Чтобы удалить контрольное значение,

  1. Перейдите на вкладку Формула.
  2. Нажмите на кнопку Окно контрольных значений.
  3. Нажмите на кнопку Удалить контрольное значение и выберите, надо ли Удалить контрольное значение или Удалить все.
