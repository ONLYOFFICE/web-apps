À propos de la Visionneuse de diagrammes

La Visionneuse de diagrammes est une application en ligne vous permettant d'afficher les diagrammes dans votre navigateur.

À l'aide de la Visionneuse de diagrammes vous pouvez imprimer les diagrammes en conservant tous les détails de mise en forme ou télécharger les diagrammes sur votre ordinateur.

Pour afficher la version actuelle du logiciel, le numéro du build et les informations de licence dans la version en ligne, cliquez sur l'icône dans la barre latérale gauche.
