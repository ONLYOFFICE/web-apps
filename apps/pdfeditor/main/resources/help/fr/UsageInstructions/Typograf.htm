Corrections typographiques

Si vous avez besoin de corriger des erreurs typographiques dans un texte, utilisez le module complémentaire Typograf permettant de placer automatiquement un espace insécable ou supprimer des espaces insécables supplémentaires ainsi que corriger les erreurs typographiques mineures, insérer des guillemets corrects, convertir des traits d'union en tirets, etc.

À partir de la version 8.2 ONLYOFFICE Docs, aucun module complémentaire n'est inclus dans l'éditeur par défaut. Il faut installer les modules complémentaires à l'aide du Gestionnaire de plugins.

  1. Accédez à l'onglet Modules complémentaires et cliquez sur Typograf.
  2. Cliquez sur le bouton Afficher les paramètres avancés.
  3. Choisissez la langue et les règles à appliquer à votre texte.
  4. Sélectionnez le texte à corriger.
  5. Cliquez sur le bouton Corriger du texte.

Pour en savoir plus d'informations sur le module complémentaire Typograf et son installation, veuillez consulter la page de modules complémentaires dans AppDirectory.
