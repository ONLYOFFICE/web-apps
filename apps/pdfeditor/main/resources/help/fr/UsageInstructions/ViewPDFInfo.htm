Afficher les informations sur le document PDF

Pour accéder aux informations détaillées sur le document PDF en cours de modification dans l'éditeur PDF, cliquez sur l'onglet Fichierde la barre d'outils supérieure et sélectionnez l'option Descriptif du document.

Informations générales

Le descriptif du document PDF comprend les propriétés afférentes à un fichier PDF. Certains de ces paramètres sont mis à jour automatiquement mais les autres peuvent être modifiés.

  • Emplacement - le dossier dans le module Documents où le fichier est stocké. Propriétaire- le nom de l'utilisateur qui a créé le fichier. Chargé- la date et l'heure quand le fichier a été créé. Ces paramètres ne sont disponibles que sous la version en ligne.
  • Statistiques- le nombre de pages, paragraphes, mots, caractères, caractères avec des espaces et taille de page.
  • Dernière modification- la date et l'heure quand le fichier a été modifié la dernière fois.
  • Applicationet Producteur PDF- l'application dans laquelle on a créé le document PDF.
  • Version PDF- la version du fichier PDF.
  • PDF marqué - affiche si le document PDF comporte les balises d'accessibilité masquées.
  • Affichage rapide sur le Web- affiche si l'affichage rapide des pages web a été activé pour ce document PDF. Ça peut s'avérer utile pour des fichiers volumineux.

Si vous avez modifié les paramètres du fichier, cliquez sur Appliquerpour enregistrer les modifications.

Remarque: Les éditeurs en ligne vous permettent de modifier le nom du document directement à partir de l'interface de l'éditeur. Pour ce faire, cliquez sur l'onglet Fichierde la barre d'outils supérieure et sélectionnez l'option Renommer, ensuite saisissez le Nom de fichierapproprié dans la nouvelle fenêtre qui s'affiche et cliquez sur OK.

Informations d'autorisation

Vous pouvez consulter les informations sur les permissions pour des fichiers stockés dans le cloud.

Remarque: cette option n'est pas disponible pour les utilisateurs disposant des autorisations en Lecture seule.

Pour savoir qui a le droit de lecture ou d'édition du document PDF, sélectionnez l'option Droits d'accès...dans la barre latérale de gauche.

Vous pouvez également modifier les droits d'accès actuels en cliquant sur le bouton Modifier les droits d'accèsdans la section Personnes qui ont des droits.

Pour fermer l'onglet Fichier et reprendre le travail sur votre document PDF, sélectionnez l'option Fermer le menu.
