Вкладка Плагины

Вкладка Плагины редактора PDF позволяет получить доступ к дополнительным возможностям редактирования, используя доступные сторонние компоненты.

Окно десктопного редактора PDF:

В настоящее время доступны следующие плагины:

  • Отправить - позволяет отправить PDF по электронной почте с помощью десктопного почтового клиента по умолчанию (доступно только в десктопной версии),
  • HTML - позволяет получить выделенный текст в виде HTML-кода.
  • Telegram можно интегрировать в рабочую область, чтобы продолжить общение в чате.
  • Подсчет слов и символов подсчитывает точное количество слов и символов в PDF-файле, как с пробелами, так и без них, а также общее количество абзацев.
  • Переводчик - позволяет переводить выделенный текст на другие языки,

    Примечание: этот плагин не работает в Internet Explorer.

  • Синонимы - позволяет осуществлять поиск синонимов и антонимов слова,
  • Речь - позволяет преобразовать выделенный текст в речь (доступно только в онлайн-версии),

Для получения дополнительной информации о плагинах, пожалуйста, обратитесь к нашей Документации по API. Все доступные примеры плагинов с открытым исходным кодом доступны на GitHub.
