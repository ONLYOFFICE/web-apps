Вкладка Комментарий

Вкладка Комментарий редактора PDF позволяет организовать совместную работу над PDF с помощью комментариев и различных инструментов разметки.

С помощью этой вкладки вы можете выполнить следующие действия:

  • подсветить цветом выделенный текст,
  • добавлять комментарии,
  • применить зачеркнутое или подчеркнутое форматирование к выделенному тексту,
  • разметить PDF с помощью одной из трех доступных ручек,
  • показать или скрыть комментарии с помощью соответствующей галочки.
