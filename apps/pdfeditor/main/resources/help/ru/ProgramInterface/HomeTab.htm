Вкладка Главная

Вкладка Главная редактора PDF открывается по умолчанию при открытии PDF. Она позволяет осуществлять навигацию по PDF, выделять текст или объекты, настраивать масштаб, устанавливать его по размеру страницы или по ширине.

С помощью этой вкладки вы можете выполнить следующие действия:

  • перемещаться по файлу,
  • настраивать масштаб,
  • устанавливать масштаб по размеру страницы или по ширине.
