Сочетания клавиш

Подсказки для клавиш

Используйте сочетания клавиш для более быстрого и удобного доступа к функциям редактора PDF без использования мыши.

  1. Нажмите клавишу Alt (Option для macOS), чтобы показать все подсказки для клавиш верхней панели инструментов, правой и левой боковой панели, а также строке состояния.
  2. Нажмите клавишу, соответствующую элементу, который вы хотите использовать. В зависимости от нажатой клавиши, могут появляться дополнительные подсказки. Когда появляются дополнительные подсказки для клавиш, первичные - скрываются.

    Например, чтобы открыть вкладку Комментарий, нажмите Alt (Option для macOS) и просмотрите все подсказки по первичным клавишам.

    Нажмите букву В, чтобы открыть вкладку Комментарий и просмотреть все доступные сочетания клавиш для этой вкладки.

    Затем нажмите букву, соответствующую элементу, который вы хотите использовать.

  3. Нажмите Alt (Option для macOS), чтобы скрыть все подсказки для клавиш, или Escape, чтобы вернуться к предыдущей группе подсказок для клавиш.

Работа с PDF
Открыть панель 'Файл' Alt+F ^ Ctrl+⌥ Option+F Открыть панель Файл, чтобы сохранить, скачать, распечатать текущий PDF-файл, просмотреть сведения о нем, создать новый документ или открыть существующий, получить доступ к Справке по редактору PDF или дополнительным параметрам.
Открыть диалоговое окно 'Поиск' Ctrl+F ^ Ctrl+F,
⌘ Cmd+F		 Открыть диалоговое окно Поиск, чтобы начать поиск символа/слова/фразы в редактируемом PDF-файле.
Открыть панель 'Комментарии' Ctrl+⇧ Shift+H ^ Ctrl+⇧ Shift+H,
⌘ Cmd+⇧ Shift+H		 Открыть панель Комментарии, чтобы добавить свой комментарий или ответить на комментарии других пользователей.
Открыть поле комментария Alt+H ⌘ Cmd+⌥ Option+A Открыть поле ввода данных, в котором можно добавить текст комментария.
Открыть панель 'Чат' (Онлайн-редакторы) Alt+Q ^ Ctrl+⌥ Option+Q Открыть панель Чат в онлайн-редакторах и отправить сообщение.
Печать PDF Ctrl+P ^ Ctrl+P,
⌘ Cmd+P		 Распечатать PDF на одном из доступных принтеров или сохранить в файл.
Скачать как... Ctrl+⇧ Shift+S ^ Ctrl+⇧ Shift+S,
⌘ Cmd+⇧ Shift+S		 Открыть панель Скачать как..., чтобы сохранить редактируемый PDF-файл на жестком диске компьютера в одном из поддерживаемых форматов: DOCX, PDF, ODT, DOTX, OTT, RTF, TXT, FB2, EPUB, HTML, JPG, PNG.
Полноэкранный режим (Онлайн-редакторы) F11 Переключиться в полноэкранный режим в онлайн-редакторах, чтобы развернуть редактор PDF на весь экран.
Меню Справка F1 Fn+F1 Открыть меню Справка редактора PDF.
Открыть существующий файл Ctrl+O ⌘ Cmd+O Открыть стандартное диалоговое окно для выбора существующего файла. Если выбрать файл в этом диалоговом окне и нажать кнопку Открыть, файл откроется в новой вкладке или в окне десктопных редакторов.
Перейти к следующей вкладке Ctrl+↹ Tab ^ Ctrl+↹ Tab Перейти к следующей вкладке файла в десктопных редакторах или вкладке браузера в онлайн-редакторах.
Перейти к предыдущей вкладке Ctrl+Shift+↹ Tab ^ Ctrl+⇧ Shift+↹ Tab Перейти к предыдущей вкладке файла в десктопных редакторах или вкладке браузера в онлайн-редакторах.
Закрыть файл Ctrl+W ⌘ Cmd+W Закрыть выбранный PDF-файл.
Закрыть меню или модальное окно, сбросить режим и др. Esc Esc Закрыть меню или модальное окно. Cбросить всплывающие окна с комментариями. Сбросить режим выделения маркером. Выйти из заполнения формы.
Сбросить масштаб Ctrl+0 ^ Ctrl+0 или ⌘ Cmd+0 Сбросить масштаб текущего PDF-файла до значения по умолчанию 100%.
Навигация
Перейти в начало строки Home ⌘ Cmd+← Установить курсор в начале редактируемой строки в поле ввода комментария.
Перейти в начало PDF-файла Ctrl+Home ^ Ctrl+Fn+←,
⌘ Cmd+Fn+←		 Перейти в самое начало редактируемого PDF-файла.
Перейти в конец строки End ⌘ Cmd+→ Установить курсор в конце редактируемой строки в поле ввода комментария.
Перейти в конец PDF-файла Ctrl+End ^ Ctrl+Fn+→,
⌘ Cmd+Fn+→		 Перейти в самый конец редактируемого PDF-файла.
Прокрутить вниз Page Down Fn+↓ Прокрутить PDF-файл примерно на одну видимую область страницы вниз.
Прокрутить вверх Page Up Fn+↑ Прокрутить PDF-файл примерно на одну видимую область страницы вверх.
Следующая страница Alt+Page Down Перейти на следующую страницу редактируемого PDF-файла.
Предыдущая страница Alt+Page Up Перейти на предыдущую страницу редактируемого PDF-файла.
Увеличить Ctrl+Num+,
Ctrl++ (на основной клавиатуре)		 ^ Ctrl++,
⌘ Cmd++		 Увеличить масштаб редактируемого PDF-файла.
Уменьшить Ctrl+- (на основной клавиатуре) ^ Ctrl+-,
⌘ Cmd+-		 Уменьшить масштаб редактируемого PDF-файла.
Выделение текста
Выделить все Ctrl+A ⌘ Cmd+A,
^ Ctrl+A		 Выделить весь текст PDF-файла.
