Redact tab

The Redact tab in the PDF Editor allows you to black out, i.e. redact, sensitive information in a .pdf.

The corresponding window of the Online PDF Editor:

The corresponding window of the Desktop PDF Editor:

Using this tab, you can mark up text for redaction in one of the following ways:

  • by manually selecting text pieces,
  • by selecting pages,
  • by finding the required text pieces via the Search tool.
