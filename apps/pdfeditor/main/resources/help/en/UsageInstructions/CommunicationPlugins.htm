Communicate while editing

In ONLYOFFICE PDF Editor, you can always keep in touch with colleagues and use popular online messengers, such as Telegram.

Telegram plugin is not installed by default. To find information on how to install it, please refer to the corresponding article: Adding plugins to ONLYOFFICE Desktop Editors Adding plugins to ONLYOFFICE Cloud, or install the plugin using the Plugin Manager.

Telegram bot

To connect ONLYOFFICE to Telegram,

  1. Follow the installation link and click the START button.

    Next time you launch the bot, simply use the /start command to access the start menu with the available options.

  2. To create and edit files:

    • Click the Create button in the menu.
    • Select the desired file format: Document, Spreadsheet, or Presentation.
    • Enter the file name.
    • The bot will send a message containing a link to the created file. Click the link to open the file in the corresponding ONLYOFFICE editor (depending on the chosen file type).
    • Once the editing session is complete, the bot will send the file with the saved changes.

      The editing link will deactivate once the modified file is sent. To reopen the file, simply reply to the message containing the file with any text.

  3. To collaborate on files:
    • To co-edit the created file, forward the message with the link to other Telegram users.
    • The link remains active for 24 hours. Each time someone clicks on it within this period, the timer resets for another 24 hours.
    • If the link has expired, an error message will appear. Once all participants who accessed the link have closed the editor, the updated file (if changes were made) will be instantly sent to everyone involved.
  4. To open a local file:
    • Send a file to the chat without selecting any command.
    • Select the following options via the bot menu: /start → Open → Upload a file from device.

      In both cases, the bot will send a message with the link to open.

      If you send several files in one message, the link will be generated for the first file in the message.

  5. To convert a file:
    • Select the following options via the bot menu: /start → Convert.
    • Send the file from your device to the bot.
    • Choose the format you want the uploaded file to be converted to. See the list of the supported formats on the GitHub page.

      Once completed, the bot will send the converted file directly to the chat in the selected format.

  6. To get some hints for working with the bot, use the /help command.

Using the Telegram bot in group chats

The bot’s functionality varies when used in a group compared to one-on-one interactions:

  • File creation is managed through three commands: /document, /spreadsheet, and /presentation. Users do not manually set the file name. Instead, it is automatically generated using the template: New [format] Date Time UTC.
  • Bot interaction varies based on admin rights. Without admin rights, you need to reply directly to the bot, and it cannot access forwarded file messages (except its own). With admin rights, the bot sees all messages, allowing you to forward files without replying directly.
  • To generate an editor link for a file anew, reply to the message containing the file and tag the bot @ONLYOFFICE_bot. In some cases, the bot may require admin access rights in the group.
  • When opening files, the bot will not respond to files simply sent to the chat. To open a file, use the /open command and reply to the bot with the file attached in a response message.
  • The bot’s language cannot be set universally for the group. Instead, it will respond in the language chosen by each individual user.

    By default, the ONLYOFFICE bot uses the language of the Telegram user profile. If such a language is not available in the bot translations, English is used. If you want to change the bot language manually, enter the /lang command.

  • File conversion is not available.
