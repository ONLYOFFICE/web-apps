You can protect your PDF forms with a password that is required to enter the editing mode by your co-authors. The password can be changed or removed later on.

The password cannot be restored if you lose or forget it. Please keep it in a safe place.

Setting a password

  1. Go to the File tab at the top toolbar and choose the Protect option. Click the Add password button to open the Set password window.

    or

    Go to the Protection tab and choose the Encrypt option.

  2. Set a password in the Password field and repeat it in the Repeat password field below. Click to show or hide password characters when entered.

  3. Click OK when ready.

Changing a password

  1. Go to the File tab at the top toolbar and choose the Protect option. Click the Change password button to open the Set password window.

    or

    Go to the Protection tab and choose the Encrypt option.

  2. Set a new password in the Password field and repeat it in the Repeat password field below. Click to show or hide password characters when entered.
  3. Click OK.

Deleting a password

  1. Go to the File tab at the top toolbar.
  2. Choose the Protect option.
  3. Click the Delete password button.
