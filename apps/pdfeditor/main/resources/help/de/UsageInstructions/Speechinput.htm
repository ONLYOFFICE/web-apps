Spracheingabe

Im PDF-Editor können Sie Text mit Ihrer Stimme einfügen.

Ab ONLYOFFICE Docs 8.2 sind standardmäßig keine Plugins mehr in den Editoren enthalten. Die Plugins müssen über die Plugin-Verwaltung installiert werden.

  1. Setzen Sie den Cursor an die Stelle, an der Sie Text hinzufügen möchten.
  2. Wechseln Sie zur Registerkarte Plugins und wählen Sie
    Spracheingabe.
  3. Wählen Sie im Popup-Fenster die Erkennungssprache aus.
  4. Klicken Sie auf die Schaltfläche Diktat starten und beginnen Sie zu sprechen. Sobald Sie pausieren, wird der Text dem Dokument hinzugefügt. Um die Spracherkennung auszuschalten, klicken Sie die Schaltfläche erneut an.
    Damit das Plugin ordnungsgemäß funktioniert, müssen Sie über ein Eingabegerät (z. B. ein Mikrofon oder Headset) sowie Berechtigungen für den Browser zur Verwendung dieser Aufnahmegeräte verfügen.
