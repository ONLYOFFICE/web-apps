Symbole

Im Dokumenteneditor, Tabellenkalkulationseditor, Präsentationseditor und PDF-Editor können Sie Tausende von Vektorsymbolen mithilfe des Plugins Symbole einfügen und anpassen. Das Plugin basiert auf der Bibliothek Font Awesome. Weitere Informationen finden Sie auf der GitHub-Seite des Plugins.

Ab ONLYOFFICE Docs 8.2 werden keine Plugins mehr standardmäßig mit den Editoren geliefert. Die Plugins müssen über die Plugin-Verwaltung installiert werden.

  1. Wechseln Sie zur Registerkarte Plugins und wählen Sie
    Symbole.
  2. Suchen Sie das gewünschte Symbol entweder über das Dropdown-Menü Kategorie oder mithilfe des Felds Suchen.
  3. Wählen Sie den gewünschten Symbolstil: Regular, Solid oder Brands.

  4. Wählen Sie das Symbol aus und klicken Sie auf Einfügen.

Eingefügte Symbole anpassen

Sobald ein Symbol eingefügt wurde, können Sie es über das Menü Formeinstellungen im rechten Seitenbereich anpassen:

  • Die Größe des Symbols ändern.
  • Die Farbe des Symbols ändern.
  • Stil und Form anpassen.
  • Das Symbol zur weiteren Bearbeitung in eine Form umwandeln.
