Registerkarte Einfügen

Die Registerkarte Einfügen im PDF-Editor ermöglicht das Hinzufügen einiger Seitenformatierungselemente sowie visueller Objekte und Kommentare.

Dialogbox Online-PDF-Editor:

Dialogbox Desktop-PDF-Editor:

Auf dieser Registerkarte können Sie:

  • Die Seite oder Objekte mit dem Hand-Werkzeug verschieben.
  • Mit dem Werkzeug Auswählen visuelle Elemente auf der Seite zur weiteren Bearbeitung auswählen.
  • Eine leere Seite in die Datei einfügen. Klicken Sie auf den Pfeil auf diesem Symbol, um auszuwählen, ob Sie eine leere Seite vor oder nach dem Cursor einfügen möchten.
  • Eine Tabelle in die Datei einfügen.
  • Ein Textfeld, Text-Art, Bild oder Form in die Datei einfügen.
  • Einen Hyperlink in die Datei einfügen.
  • Eine Gleichung oder ein Symbol in die Datei einfügen.
