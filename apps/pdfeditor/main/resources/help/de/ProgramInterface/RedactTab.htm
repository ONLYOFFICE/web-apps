Registerkarte Schwärzen

Über die Registerkarte Schwärzen im PDF-Editor können Sie vertrauliche Informationen in einer PDF-Datei schwärzen, d. h. redigieren.

Das entsprechende Fenster des Desktop-PDF-Editors:

Mithilfe dieser Registerkarte können Sie Text zum Schwärzen markieren und zwar auf eine der folgenden Arten:

  • durch manuelles Auswählen von Textteilen,
  • durch Auswählen von Seiten,
  • indem Sie die erforderlichen Textteile über das Suchtool finden.
