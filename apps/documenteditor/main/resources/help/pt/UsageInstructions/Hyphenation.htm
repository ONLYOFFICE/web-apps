Adicionar hifenização

A hifenização é usada para separar palavras no final das linhas com um sinal de "-" de forma linguisticamente correta.

Para adicionar hifenização no Editor de Documentos:

  1. Alterne para a guia Layout da barra de ferramentas superior.
  2. Clique no ícone
    Hifenização na barra de ferramentas superior e escolha o item de menu Automático para aplicar a hifenização ao documento.
  3. Para acessar as configurações de hifenização, clique na seta abaixo do ícone
    Hifenização, e selecione o item de menu Opções de hifenização:

    • Hifenizar automaticamente o documento - marque esta caixa para hifenizar o documento.
    • Hifenizar palavras em maiúsculo - desmarque essa caixa para que as palavras escritas em maiúsculo sejam ignoradas durante a hifenização.
    • Zona de hifenização - insira um valor manualmente para controlar a distância da borda da página em que um hífen é necessário.
    • Limite os hífens consecutivos a - insira um valor manualmente para controlar o número de hífens presentes consecutivamente ou deixe a caixa inalterada com uma opção padrão Sem limites.
  4. Clique no botão OK.

Para excluir a hifenização, clique na seta abaixo do ícone Hifenização, e selecione a opção Nenhum.
