Pixabay

O plugin Pixabay permite adicionar imagens ao seu documento a partir de uma coleção aberta do serviço Pixabay, que oferece imagens livres de direitos autorais.

O plug-in é compatível com as versões auto-hospedadas e para computadores dos editores ONLYOFFICE e pode ser adicionado manualmente às instâncias ONLYOFFICE usando o Gerenciador de Plug-ins.

Instalação

Para instalar o plug-in Pixabay,

  1. Vá para a guia Plug-ins.
  2. Abra o Gerenciador de Plug-ins.
  3. Encontre
    no marketplace e clique no botão Instalar abaixo.
  4. Clique no ícone
    Pixabay na guia Plug-ins.
  5. Prossiga com a configuração do plug-in.

Para obter mais detalhes, consulte a documentação do ONLYOFFICE API.

Configuração

  1. Faça login em sua conta do Pixabay ou registre uma nova.
  2. Vá para a seção Pesquisar Imagens da página Pixabay API.
  3. Role para baixo até a lista Parâmetros e copie o link abaixo do parâmetro Chave. Se você não estiver conectado, pressione Login próximo ao parâmetro Chave.

  4. Cole a chave no campo chave de API no painel esquerdo da guia Plug-ins no Editor de Documentos.
  5. Clique em Salvar.

Como usar

  1. Vá para a guia Plug-ins.
  2. Clique no ícone
    Pixabay.
  3. No painel esquerdo que se abre, digite a palavra-chave que você associa à imagem que está procurando.
  4. Use o botão Mostrar configurações para refinar sua pesquisa por Idioma, Tipo de imagem, Orientação e Categoria. Clique em Esconder configurações para reduzir a seção de pesquisa.

  5. Clique em Pesquisar.
  6. Percorra os resultados da pesquisa e clique na imagem para adicioná-la ao seu documento.
