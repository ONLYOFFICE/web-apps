Definitionen einfügen

Im Dokumenteneditor können Sie mit dem Plugin TerMef nach Begriffen suchen, deren Definitionen aus der Datenbank anzeigen und die gewünschte Definition als Endnote am Ende Ihres Dokuments einfügen. Das Plugin funktioniert nur mit Begriffen und Definitionen in französischer Sprache. Es kann über den integrierten Plugin-Manager installiert werden.

Ab ONLYOFFICE Docs 8.2 sind standardmäßig keine Plugins mehr in den Editoren enthalten. Die Plugins müssen über die Plugin-Verwaltung installiert werden.

  1. Wählen Sie Ihren Begriff im Dokument aus.
  2. Markieren Sie die erforderliche Definition im linken Bereich.
  3. Fügen Sie die Definition als Endnote ein, indem Sie auf die Schaltfläche Endnote klicken.
