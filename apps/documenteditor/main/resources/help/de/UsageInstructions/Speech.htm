Text laut vorlesen

ONLYOFFICE Dokumenteneditor hat ein Plugin, das den Text für Sie vorlesen kann.

Ab ONLYOFFICE Docs 8.2 sind standardmäßig keine Plugins mehr in den Editoren enthalten. Die Plugins müssen über die Plugin-Verwaltung installiert werden.

  1. Wählen Sie den vorzulesenden Text aus.
  2. Öffnen Sie die Registerkarte Plugins und wählen Sie den Menüpunkt
    Rede aus.

Der Text wird nun vorgelesen.
