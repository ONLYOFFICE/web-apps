Textformatierung übernehmen/entfernen

Kopieren einer bestimmte Textformatierung im Dokumenteneditor:

  1. Wählen Sie mit der Maus oder mithilfe der Tastatur den Textabschnitt aus, dessen Formatierung Sie kopieren möchten.
  2. Klicken Sie in der oberen Symbolleiste unter der Registerkarte Startseite auf das Symbol
    Format übertragen (der Mauszeiger ändert sich wie folgt
    ).
  3. Wählen Sie einen Textabschnitt, auf den Sie die Formatierung übertragen möchten.

Übertragung der Formatierung auf mehrere Textabschnitte:

  1. Wählen Sie mit der Maus oder mithilfe der Tastatur den Textabschnitt aus, dessen Formatierung Sie kopieren möchten.
  2. Führen Sie in der oberen Symbolleiste unter der Registerkarte Startseite einen Doppelklick auf das Symbol
    Format übertragen aus (der Mauszeiger ändert sich wie folgt
    und das Symbol Format übertragen bleibt ausgewählt:
    ).
  3. Markieren Sie die gewünschten Textabschnitte Schritt für Schritt, um die Formatierung zu übertragen.
  4. Wenn Sie den Modus beenden möchten, klicken Sie erneut auf das Symbol Format übertragen
    oder drücken Sie die ESC-Taste auf Ihrer Tastatur.

Wenn Sie die angewandte Formatierung wieder entfernen möchten,

  1. markieren Sie den entsprechenden Textabschnitt und
  2. klicken Sie auf der oberen Symbolleiste, unter der Registerkarte Start auf das Symbol Formatierung löschen
    .
