Hervorgehobenen Code einfügen

Im Dokumenteneditor können Sie hervorgehobenen Code mit dem schon angepassten Stil entsprechend der Programmiersprache und dem Farbstil des von Ihnen ausgewählten Programms einfügen.

  1. Gehen Sie zu Ihrem Dokument und platzieren Sie den Cursor an der Stelle, an der Sie den Code einfügen möchten.
  2. Öffnen Sie die Registerkarte Plugins und wählen Sie den Menüpunkt Code hervorheben aus.
  3. Geben Sie die Programmiersprache an.
  4. Wählen Sie einen Code-Stil aus, der so aussieht, als wäre er in diesem Programm geöffnet.
  5. Geben Sie an, ob Sie Tabulatoren durch Leerzeichen ersetzen möchten.
  6. Wählen Sie Hintergrundfarbe. Bewegen Sie dazu den Cursor manuell über die Palette oder fügen Sie den RGB/HSL/HEX-Wert ein.
  7. Klicken Sie auf OK, um den Code einzufügen.
