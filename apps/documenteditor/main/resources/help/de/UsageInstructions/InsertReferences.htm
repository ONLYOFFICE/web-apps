Verweise einfügen

ONLYOFFICE Dokumenteneditor unterstützt die Literaturverwaltungsprogramme wie Mendeley und Zotero beim Einfügen von Quellenangaben in Ihr Dokument.

Ab ONLYOFFICE Docs 8.2 sind standardmäßig keine Plugins mehr in den Editoren enthalten. Die Plugins müssen über den Plugin-Manager installiert werden.

Mendeley

ONLYOFFICE mit Mendeley verbinden

  1. Melden Sie sich bei Ihrem Mendeley-Konto an.
  2. Wechseln Sie in Ihrem Dokument zur Registerkarte Plugins und wählen Sie
    Mendeley aus. Eine Seitenleiste wird links in Ihrem Dokument geöffnet.
  3. Klicken Sie auf die Schaltfläche Link kopieren und Formular öffnen.
    Der Browser öffnet ein Formular auf der Mendeley-Seite. Füllen Sie dieses Formular aus und notieren Sie die Anwendungs-ID für ONLYOFFICE.
  4. Wechseln Sie zurück zu Ihrem Dokument.
  5. Geben Sie die Anwendungs-ID ein und klicken Sie auf Speichern.
  6. Klicken Sie auf Anmelden.
  7. Klicken Sie auf Weiter.

Jetzt ist ONLYOFFICE mit Ihrem Mendeley-Konto verbunden.

Verweise einfügen

  1. Öffnen Sie das Dokument und platzieren Sie den Cursor an der Stelle, an der Sie den Verweis(-e) einfügen möchten.
  2. Öffnen Sie die Registerkarte Plugins und wählen Sie den Menüpunkt
    Mendeley aus.
  3. Geben Sie einen Suchtext ein und drücken Sie die Eingabetaste auf Ihrer Tastatur.
  4. Klicken Sie auf ein oder mehrere Kontrollkästchen.
  5. [Optional] Geben Sie einen neuen Suchtext ein und klicken Sie auf ein oder mehrere Kontrollkästchen.
  6. Wählen Sie den Referenzstil aus dem Pulldown-Menü Stil.
  7. Klicken Sie auf die Schaltfläche Bibliographie einfügen.

Zotero

ONLYOFFICE mit Zotero verbinden

  1. Melden Sie sich bei Ihrem Zotero-Konto an.
  2. Wechseln Sie in Ihrem Dokument zur Registerkarte Plugins und wählen Sie
    Zotero aus. Eine Seitenleiste wird links in Ihrem Dokument geöffnet.
  3. Klicken Sie auf Zotero API-Einstellungen.
  4. Erstellen Sie auf der Zotero-Seite einen neuen Schlüssel für Zotero, kopieren Sie ihn und speichern Sie ihn für später.
  5. Wechseln Sie zu Ihrem Dokument und fügen Sie den API-Schlüssel ein.
  6. Klicken Sie auf Speichern.

Jetzt ist ONLYOFFICE mit Ihrem Zotero-Konto verbunden.
