Text aus Datei einfügen

Im Dokumenteneditor können Sie Text aus einer Datei einfügen und dabei die Formatierung beibehalten.

  1. Wechseln Sie zur Registerkarte Einfügen in der oberen Symbolleiste.

  2. Klicken Sie auf das Symbol
    Text aus Datei.
  3. Wählen Sie eine der folgenden Optionen:
    • Text aus der lokalen Datei: Wählen Sie eine Datei auf Ihrer Festplatte aus, um sie in das Dokument einzufügen.
    • Text aus der URL-Datei: Geben Sie die erforderliche URL ein, um den Text daraus einzufügen. Bitte beachten Sie, dass es sich dabei um eine öffentlich zugängliche Textdatei handeln muss, damit der Vorgang ordnungsgemäß ausgeführt werden kann.
    • Text aus der Ablagedatei: Wählen Sie eine Datei aus Ihrem ONLYOFFICE-Portal aus, um deren Inhalt einzufügen.
