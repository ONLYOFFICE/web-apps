Расстановка переносов

Расстановка переносов используется для лингвистически правильного разделения слов в конце строк знаком «-».

Чтобы расставить переносы в Редакторе документов:

  1. Перейдите на вкладку Макет верхней панели инструментов.
  2. Щелкните значок
    Расстановка переносов на верхней панели инструментов и выберите пункт меню Автоматически, чтобы применить расстановку переносов к документу.
  3. Чтобы получить доступ к параметрам расстановки переносов, щелкните стрелку под значком
    Расстановка переносов и нажмите Параметры расстановки переносов:

    • Автоматическая расстановка переносов — установите флажок, чтобы расставлять переносы в документе.
    • Переносы в словах из ПРОПИСНЫХ БУКВ — снимите флажок, чтобы при расстановке переносов игнорировать слова, написанные ЗАГЛАВНЫМИ БУКВАМИ.
    • Ширина зоны переноса слов — введите значение расстояние от края страницы, при котором будет расставляться дефис.
    • Макс. число последовательных переносов — введите значение количества последовательных дефисов или оставьте выбранный по умолчанию параметр Нет, чтобы производить переносы без ограничений.
  4. Нажмите кнопку OK.

Чтобы удалить расстановку переносов, щелкните стрелку под значком Расстановка переносов и нажмите Нет.
