Insérer des références

L'éditeur de documents ONLYOFFICE prend en charge les logiciels de gestion de références Mendeley et Zotero pour insérer des références dans votre document.

À partir de la version 8.2 ONLYOFFICE Docs, aucun module complémentaire n'est inclus dans l'éditeur par défaut. Il faut installer les modules complémentaires à l'aide du Gestionnaire de plugins.

Mendeley

Connecter ONLYOFFICE à Mendeley

  1. Connectez-vous à votre compte Mendeley.
  2. Dans votre document, passez à l'onglet Modules complémentaires et choisissez
    Mendeley, un panneau s'ouvrira à gauche de votre document.
  3. Cliquez sur le bouton Copier le lien et ouvrir le formulaire.
    Le navigateur ouvre le formulaire sur le site Mendeley. Remplissez le formulaire et notez l'identifiant de l'application ONLYOFFICE.
  4. Revenez à votre document.
  5. Saisissez l'identifiant de l'application et cliquez sur le bouton Enregistrer.
  6. Cliquez sur Se connecter.
  7. Cliquez sur Continuer.

Désormais, ONLYOFFICE est connecté à votre compte Mendeley.

Insérer des références à l'aide de Mendeley

  1. Accédez à votre document et placez le curseur à l'endroit où vous souhaitez insérer une ou plusieurs références.
  2. Passez à l'onglet Modules complémentaires et sélectionnez
    Mendeley.
  3. Saisissez le texte à rechercher et appuyez sur la touche Entrée du clavier.
  4. Activez une ou plusieurs cases à cocher.
  5. (Facultatif) Saisissez le nouveau texte à rechercher et activez une ou plusieurs cases à cocher.
  6. Sélectionnez le style de la référence de la liste déroulante Style.
  7. cliquez sur le bouton Insérer la bibliographie.

Zotero

Connecter ONLYOFFICE à Zotero

Le module complémentaire Zotero permet d'ajouter des citations et des références bibliographiques dans ONLYOFFICE Docs. Vous pouvez travailler sur votre bibliothèque Zotero en ligne ou utiliser vos données Zotero locales hors connexion.

Nouveautés de la version 1.0.6

  • Interface actualisée pour une meilleure convivialité
  • Possibilité de choisir la bibliothèque à rechercher
  • Possibilité d'ajouter un préfixe et un suffixe
  • Fonctionnalité d'omission de l'auteur Omit author
  • Prise en charge des localisateurs de citation : On a ajouté la possibilité d'inclure des numéros de page, des chapitres, des sections et d'autres références de localisation dans les citations

Vous pouvez consulter le journal des modifications complet ici.

Comment connecter le module complémentaire

Vous pouvez connecter le plugin de deux façons:

  • en utilisant votre compte Zotero en ligne (clé API), ou
  • en utilisant Zotero Desktop pour un accès hors connexion.

Connexion en ligne (méthode API)

Utilisez cette méthode si vous souhaitez accéder à votre bibliothèque Zotero synchronisée dans le cloud :

  1. Accédez à Modules complémentaires →
    Zotero.
  2. Cliquez sur le bouton Se connecter avec une clé API.
  3. Si vous ne disposez pas de clé API, vous pouvez vous enregistrer sur le site officiel Zotero et générer la clé (Paramètres → Sécurité → Nouvelle clé).
  4. Collez la clé dans le champ du Module complémentaire.
  5. Votre bibliothèque Zotero en ligne se chargera dans le panneau de gauche.

Connexion hors connexion (méthode Zotero locale)

Utilisez cette méthode si vous souhaitez accéder à votre bibliothèque Zotero locale sans connexion Internet :

  1. Installez Zotero Desktop.
  2. Dans Zotero, passez à Modifier → Paramètres → Avancés.
  3. Activez l'option Autoriser d'autres applications sur cet ordinateur à communiquer avec Zotero.
  4. Accédez à l'onglet Synchronisation dans le panneau de gauche → connectez-vous utilisant vos identifiants Zotero → cliquez sur Configurer la synchronisation.
  5. Ouvrez l'éditeur de bureau ONLYOFFICE.
  6. Passez à Modules complémentaires →
    Zotero.
  7. Cliquez sur Travailler avec Zotero local.
  8. Assurez-vous que Zotero Desktop est en cours d'exécution.
  • La connexion Internet n'est pas requise une fois la bibliothèque locale disponible.
  • Tous les outils de citation et de bibliographie fonctionnent en mode hors connexion.

Insérer des références à l'aide de Zotero

  1. Accédez à votre document et placez le curseur à l'endroit où vous souhaitez insérer une ou plusieurs références.
  2. Passez à l'onglet Modules complémentaires et sélectionnez
    Zotero.
  3. Saisissez le texte à rechercher et appuyez sur la touche Entrée du clavier. Cliquez sur le bouton Rechercher dans situé à côté du champ de recherche pour définir les bibliothèques.
  4. Activez une ou plusieurs cases à cocher.
  5. (Facultatif) Saisissez le nouveau texte à rechercher et activez une ou plusieurs cases à cocher.
  6. Cliquez sur l'icône de l'outil Paramètres pour sélectionner le style de citation dans le menu déroulant et sélectionnez la langue. Cliquez sur Terminé pour valider.

    Sélectionnez l'élément de menu Plus de styles dans le menu déroulant pour ouvrir la liste de tous les styles disponibles.

  7. cliquez sur le bouton Insérer la bibliographie.

Ajouter des styles de citation personnalisés

Pour installer un style personnalisé :

  1. Passez à l'onglet Modules complémentaires et choisissez
    Zotero.
  2. Cliquez sur Paramètres → ouvrez le menu déroulant → Ajouter un style personnalisé.
  3. Importez votre fichier .csl.
  4. Zotero ajoutera le style à la liste des formats de citation disponibles.
  5. Redémarrez l'éditeur pour vous assurer que le style se charge correctement.

Si vous n'avez pas de fichier .csldisponible, vous pouvez choisir parmi plus de 10 000 styles supplémentaires disponibles dans le dépôt de styles Zotero.
