Insérer des codes de champ

Dans l'éditeur de documents, vous pouvez ajouter des champs à contenu dynamique en fonction de vos informations système ou d'autre contenu. Veuillez ne pas confondre des champ de contrôles de contenu car ce sont des fonctionnalités différentes.

Pour ajouter un champ,

  1. Passez à l'onglet Insertion,
  2. Cliquez sur le bouton Champ.
  3. Dans la boîte de dialogue, saisissez le code de champ à afficher et cliquez sur OK.

Les codes de champ sont les suivants:

Code de champ ONLYOFFICE Description Syntaxe Switches
MERGEFIELD Ce champ insère un lien vers des données provenant d'une source externe dans le document actuel. Lors de la fusion et publipostage, le contenu de la source externe remplacera le code de champ. { MERGEFIELD FieldName [ Switches ]}
  1. \b - Insère du texte avant le champ MergeField si ce champ n'est pas vide.
  2. \f - Insère du texte après le champ MergeField si ce champ n'est pas vide.
  3. \m - Marque le champ MergeField comme étant mappé.
  4. \v - permet la conversion des caractères en mise en forme verticale.
PAGENUM Ce code insère le numéro de page actuel.
PAGECOUNT Ce code insère le nombre total de pages dans le document actuel.
FORMTEXT
TOC Ce code crée une table des matières. La table des matières est basée sur les styles de titre, les niveaux de plan, les légendes et d'autres éléments dans des champs de saisie de la Table de matières. { TOC [Switches ] }
  1. \a - génère la liste des éléments étiquettes et nombres omis.
  2. \b - limite la table des matières à une section du document marquée d'un signet.
  3. \с - génère la liste des éléments spécifiques.
  4. \d - spécifie le caractère qui sépare les numéros de page.
  5. \f - génère une table à partir de champs TC et non sur les niveaux hiérarchiques du document.
  6. \h - applique à la table des matières la mise en forme du lien hypertexte.
  7. \l - définit les niveaux nécessaires des champs TC. Les niveaux qui ne sont pas inclus dans la plage spécifiée sont ignorés.
  8. \n - génère la table des matières complète ou utilisant les éléments de la plage spécifiée (par exemple, "1-9"), les numéros de page sont ignorés.
  9. \o - génère une table des matières en fonction des niveaux de structure du document, en ignorant les champs TC personnalisés.
  10. \p - spécifie le caractère qui sépare un élément et son numéro de page.
  11. \s - inclut un numéro de séquence avant le numéro de page.
  12. \t - utilise des styles personnalisés pour les entrées de la table au lieu des styles de titre par défaut.
  13. \u - génère une table des matières à partir de paragraphes dont la mise en forme inclut des niveaux hiérarchiques appliqués.
  14. \w - conserve la tabulation dans les entrées de table.
  15. \x - conserve les sauts de ligne dans les entrées de table.
  16. \z - masque le numéro de page en mode document web.
PAGEREF Ce code insère le numéro de page d'un signet pour une référence croisée. { PAGEREF Bookmark [\* Format Switch ] }
  1. \h - crée un lien hypertexte au paragraphe défini.
  2. \p - fait afficher la position du paragraphe défini par rapport au signet.
PAGE Ce code insère le nombre de la page actuelle. { PAGE [\* Format Switch ] }
NUMPAGES Ce code insère le nombre total de pages dans le document. { NUMPAGES }
ASK Ce code insère le champ vous invitant à saisir des informations, qu'il attribue ensuite à signet. L'éditeur invite l'utilisateur à fournir une réponse chaque fois qu'on met à jour ce champ ou lors du publipostage. { ASK Bookmark"Prompt" [Optional switches ] }
  1. \d - insère le texte de remplacement par défaut en tant que signet.
  2. \o - invite une fois lors de la fusion des documents.
REF Ce code insère le contenu du signet. Il faut ajouter un signet au document. { [REF] Bookmark [Switches ] }
  1. \d - spécifie les caractères qui séparent les numéros de page.
  2. \f - incrémente les numéros de note de bas de page ou d'annotation et insère les notes au document.
  3. \h - crée un lien hypertexte au paragraphe défini.
  4. \n - ajoute les numéros de paragraphe, le cas échéant.
  5. \p - fait afficher la position du paragraphe défini par rapport au signet.
  6. \r - insère le numéro de paragraphe dans un contexte relatif.
  7. \t - force à supprimer des symboles non numériques ou non délimiteur.
  8. \r - insère le numéro de paragraphe entier.
HYPERLINK Ce code insère un lien hypertexte vers un autre élément dans le document, un document sur votre ordinateur, portail ou une source externe. { HYPERLINK "Filename" [Switches ] }
  1. \l - indique l'élément dans le document où ce lien hypertexte saute.
  2. \m - ajoute des coordonnées à un lien hypertexte pour un image interactive sur le serveur.
  3. \n - provoque l'ouverture de la fin de document dans une nouvelle fenêtre.
  4. \o - indique le texte de l'info-bulle.
  5. \t - indique la cible vers laquelle le lien doit être redirigé.
TIME Ce code insère l'heure actuelle dans le document en fonction des paramètres système de l'utilisateur. { TIME [\@ "Date-Time Picture"] }
DATE Ce code insère la date actuelle dans le document en fonction des paramètres système de l'utilisateur. { DATE [ \@ "Date-Time Picture"] [Switches] }
FORMULA Ce code calcule un nombre à l'aide d'une formule mathématique. { = Formula [Bookmark ] [\#Numeric Picture ] }
SEQ Ce code numérote les chapitres, les tableaux, les figures et d'autres éléments dans l'ordre. Si vous ajoutez, supprimez ou déplacez un élément et son champ Seq respectif, vous pouvez mettre à jour d'autres champs Seq dans le document pour afficher la nouvelle séquence. { SEQ Identifier [Bookmark ] [Switches ] }
  1. \c - insère le numéro de séquence précédent le plus proche.
  2. \h - masque le résultat du champ (pour référence croisée).
  3. \n - insère le numéro de séquence suivant.
  4. \r - réinitialise le numéro de séquence au nombre spécifié.
  5. \s - réinitialise le numéro de séquence au niveau hiérarchique spécifié.
STYLEREF Ce code insère du texte mis en forme avec le style spécifié. Lorsqu'il est inséré dans un en-tête ou un pied de page, le champ affiche le premier ou le dernier texte mis en forme avec le style spécifié dans le corps du document de la page en cours, ce qui vous permet de créer des en-têtes ou des pieds de page de style dictionnaire { STYLEREF StyleIdentifier [Switches ] }
  1. \l - démarre la recherche dans la page de bas en haut.
  2. \n - affiche le numéro du paragraphe spécifié.
  3. \p - fait afficher la position du paragraphe défini par rapport au signet.
  4. \r - insère le numéro de paragraphe dans un contexte relatif.
  5. \t - force à supprimer des symboles non numériques ou non délimiteur.
  6. \r - insère le numéro de paragraphe entier.
NOTEREF Ce code insère une note de bas de page ou une marque de référence de note de fin que vous avez marquée avec un signet pour faire références à la même note ou à des notes de bas de page ou de fin de page de référence croisée. Si vous modifiez la séquence de notes, le résultat du champ affiche la nouvelle numérotation. { NOTEREF Bookmark [Switches] }
  1. \f - insère la marque de référence avec la mise en forme de caractères Style référence de note de bas de page ou Style référence de note de fin.
  2. \h - insère un lien hypertexte à la note de bas de page ou à la note de fin.
  3. \p - insère la position relative de la note de bas de page ou de la note de fin.
