Icônes

Dans l'éditeur de documents, l'éditeur de classeurs, l'éditeur de présentations et l'éditeur de PDF,vous pouvez ajouter et personnaliser des milliers d'icônes vectorielles à l'aide du module complémentaire Icônes. Ce module complémentaire est basé sur la bibliothèque Font Awesome. Veuillez consulter la page sur GitHub du module complémentaire pour en savoir plus.

À partir de la version 8.2 ONLYOFFICE Docs, aucun module complémentaire n'est inclus dans l'éditeur par défaut. Il faut installer les modules complémentaires à l'aide du Gestionnaire de plugins.

  1. Passez à l'onglet Modules complémentaires et choisissez
    Icônes.
  2. Recherchez l'icône à ajouter utilisant le menu déroulant Catégorie ou la barre de Recherche.
  3. Sélectionnez le style d'icône approprié: Normal, Uni ou Brands.

  4. Sélectionnez l'icône et cliquez sur Insérer.
Personnaliser les icônes

Une fois l'icône ajoutée, vous pouvez la personnaliser à l'aide du menu Paramètres de la forme sur le volet droit:

  • Modifier la taille de l'icône.
  • Modifier la couleur de l'icône.
  • Configurer le style et la forme.
  • Convertissez l'icône en forme pour rendre possible la modification de celle-ci.
