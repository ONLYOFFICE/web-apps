Layout tab

The Layout tab of the Document Editor allows changing the appearance of a document: setting up page parameters and defining the arrangement of visual elements.

The corresponding window of the Desktop Document Editor:

Using this tab, you can:

  • adjust page margins, orientation and size,
  • add columns,
  • insert page breaks, section breaks and column breaks,
  • insert line numbers
  • align and arrange objects (tables, pictures, charts, shapes),
  • change the wrapping style and edit wrap boundary,
  • add a watermark.
