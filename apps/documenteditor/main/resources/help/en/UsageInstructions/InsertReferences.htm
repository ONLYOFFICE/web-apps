Insert references

ONLYOFFICE Document Editor supports Mendeley and Zotero reference managers to insert references into your document.

Starting with ONLYOFFICE Docs 8.2, no plugins come with the editors by default. The plugins shall be installed via Plugin Manager.

Mendeley

Connect ONLYOFFICE to Mendeley

  1. Log in to your Mendeley account.
  2. In your document, switch to the Plugins tab and choose
    Mendeley; a sidebar will open on the left side of your document.
  3. Click the Copy Link and Open Form button.
    The browser opens a form on the Mendeley site. Complete this form and note the Application ID for ONLYOFFICE.
  4. Switch back to your document.
  5. Enter the Application ID and click Save.
  6. Click Login.
  7. Click Proceed.

Now ONLYOFFICE is connected to your Mendeley account.

Inserting references using Mendeley

  1. Open the document and place the cursor on the spot where you want to insert the reference(s).
  2. Switch to the Plugins tab and choose
    Mendeley.
  3. Enter a search text and hit Enter on your keyboard.
  4. Click one or more checkboxes.
  5. (Optional) Enter a new search text and click on one or more checkboxes.
  6. Choose the reference style from the Style pull-down menu.
  7. Click the Insert Bibliography button.

Zotero

Connect ONLYOFFICE to Zotero

The Zotero plugin allows inserting citations and bibliographies in ONLYOFFICE Docs. You can work with your online Zotero library or use your local Zotero data offline.

  • The ability to choose which library to search in
  • Ability to add prefix and suffix
  • Omit author feature
  • Citation locators support: Added ability to include page numbers, chapters, sections, and other location references in citations

How to connect the plugin

You can connect the plugin in two ways:

  • using your Zotero online account (API key), or
  • using Zotero Desktop for offline access.

Connect online (API method)

Use this method if you want to access your cloud-synced Zotero library:

  1. Open Plugins →
    Zotero.
  2. Click the Log in with API key button.
  3. If you do not have an API key, you can sign up on the official Zotero website and generate one (Settings → Security → New key).
  4. Paste the key into the Plugin field.
  5. Your online Zotero library will load in the left-hand panel.

Connect offline (local Zotero method)

Use this method if you want to access your local Zotero library without an Internet connection:

  1. Install Zotero Desktop.
  2. In Zotero, go to Edit → Settings → Advanced.
  3. Enable Allow other applications on this computer to communicate with Zotero.
  4. Go to the Sync tab on the left panel → login with your Zotero credentials → click Set Up Syncing.
  5. Open ONLYOFFICE Desktop Editors.
  6. Go to Plugins →
    Zotero.
  7. Click Work with local Zotero.
  8. Make sure Zotero Desktop is running.
  • Internet is not required once the local library is available.
  • All citation and bibliography tools work in offline mode.

Inserting references using Zotero

  1. Open the document and place the cursor on the spot where you want to insert the reference(s).
  2. Switch to the Plugins tab and choose
    Zotero.
  3. Enter a search text and hit Enter on your keyboard. Click the Search in button next to the search field to set the libraries.
  4. Click one or more checkboxes.
  5. (Optional) Enter a new search text and click on one or more checkboxes.
  6. Choose the Settings tool icon to select the citation style from the drop-down menu and set the language. Click Done when ready.

    Select the More styles menu item in the drop-down menu to open up the list of all available styles.

  7. Click the Insert Bibliography button.

Adding custom citation styles

To install a custom style:

  1. Switch to the Plugins tab and choose
    Zotero.
  2. Click the Settings tool button → open the drop-down menu → Add custom style.
  3. Import your .csl file.
  4. Zotero will add the style to the list of available citation formats.
  5. Restart the editor to ensure the style loads correctly.

If you don’t have a .csl file ready, you can choose from over 10,000 additional styles available in the Zotero Style Repository.
