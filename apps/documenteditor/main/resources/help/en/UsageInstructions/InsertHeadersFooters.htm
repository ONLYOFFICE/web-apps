Insert headers and footers

To add or remove a new header/footer, or edit one that already exists Document Editor,

  1. Switch to the Insert tab of the top toolbar.
  2. Click the
    Header/Footer icon on the top toolbar.
  3. Select one of the following options:
    • Edit Header to insert or edit the header text.
    • Edit Footer to insert or edit the footer text.
    • Remove Header to delete header.
    • Remove Footer to delete footer.
  4. The Header/Footer tab will open on the top toolbar.

    You can also double-click anywhere on the top or bottom margin of your document or click there with the right mouse button and select one of the available options, Edit Header or Edit Footer correspondingly, to enter a text or edit the already entered text and adjust the header or footer settings.

  5. The following settings are available:

    • Adjust page numbering using the Page Number button.
    • Insert Date & Time.
    • Insert field codes using the Field button.
    • Insert images using the Image button.
    • Adjust the positioning of the header or the footer by manually entering the distancing values in the Header from top and Footer from bottom fields.
    • Use the Different odd and even pages box to add different headers/footers for odd and even pages.
    • Check the Different first page box to apply a different header or footer to the very first page, or in case you don't want to add any header/footer to it.
    • The Link to Previous option is available in case you've previously added sections into your document. If not, it will be grayed out. Moreover, this option is also unavailable for the very first section (i.e., when a header or footer that belongs to the first section is selected). By default, this box is checked so that the same headers/footers are applied to all the sections. If you select a header or footer area, you will see that the area is marked with the Same as Previous label. Uncheck the Link to Previous box to use different headers/footers for each section of the document. The Same as Previous label will no longer be displayed.

To switch to the document body, double-click within the working area. The text you use as a header or footer will be displayed in gray.
